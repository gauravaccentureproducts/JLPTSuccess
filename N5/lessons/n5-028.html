〜の

EN〜の (possessive / noun-modifier)

JA「Aの B」(しゅうしょく)

Explanation

の links nouns. Can mean possession (わたしの = 'my'), classification (日本語の = 'Japanese (language) X'), or general modification.

Attaches to

noun

Examples

JAこれは わたしの 本です。
ENThis is my book.
JA日本語の 先生です。
ENI'm a teacher of Japanese.
JAあれは だれの かばんですか。
ENWhose bag is that?
JAがっこうの となりの こうえんで あそびました。
ENI played in the park next to the school.
JAあの かたは 先生です。
ENThat person is a teacher. (polite)
JA父は 先生です。
ENMy father is a teacher.
JAあの 男の 人は 先生です。
ENThat man is a teacher.
JAにほんの たべものが すきです。
ENI like Japanese food.
JAやまださんの くるまは あれです。
ENYamada-san's car is that one.
JAがっこうの まえに あいましょう。
ENLet's meet in front of the school.

Common mistakes

✗ 日本語を 話す が すきです。
✓ 日本語を 話すのが すきです。
Bare verbs cannot be subjects of すき. Add の (or こと) to nominalize: 話すのが すき.
✗ これは わたしが 本です。
✓ これは わたしの 本です。
の marks possession between two nouns: A の B = 'B of A'. が is for new-information subject — would make 'as for this, I (as new info) am a book' — nonsense. For 'my book', use わたし + の + 本.
✗ これは わたしと 本です。
✓ これは わたしの 本です。
Possession between two nouns uses の (A の B = 'A's B'). と would mean 'A AND B' (coordination), not possession. For 'my book', insert の between わたし and 本.

Wrong / corrected pairs

✗ これは わたしの です ほん。
✓ これは わたしの ほんです。
Word order: possessive の + N + です. Cannot split.
✗ にほんごの ほんの です。
✓ にほんごの ほんです。
Single の per modifier; do not double.
✗ わたし ほん。 (no の)
✓ わたしの ほん。
Possessive linkage requires の: A + の + B = "A's B".
