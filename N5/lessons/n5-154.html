もう + Verb-ました

ENもう + Verb-ました - already (action completed)

JA「もう + Verb-ました」で、すでに した ことを いいます。「もう 食べました」「もう しゅくだいを しました」。よく 「まだ + Verb-ていません」(まだ 食べて いません) と ペアで つかいます。

Explanation

もう + ました = already done. Often paired with まだ + ていません as a yes/no answer pair.

Attaches to

verb_mashita

JAもう ばんごはんを たべました。
ENI've already eaten dinner.

ENHave you finished your homework yet?

ENI already went to school.

ENIs the test already over?
JAもう しゅくだいは おわりました。

ENI watched a movie with my mother.

Common mistakes

✗ もう 食べます。
✓ もう 食べました。
もう = "already" requires past (〜ました). Present tense もう食べます would mean "I'll eat now", not "already ate".
✗ もう ばんごはんを たべます。
✓ もう ばんごはんを たべました。
もう + V-ました = 'already done'. The non-past たべます would mean 'I'll eat now' — not 'I already ate'. For the perfect-aspect 'already', use もう + past polite. もう食べました = I've already eaten.
✗ もう ばんごはんを たべるました。
✓ もう ばんごはんを たべました。
Group 2 verb たべる: ます-form drops る + ます → たべます → past たべました. The form たべるました wrongly keeps the dict-form る.

Wrong / corrected pairs

✗ もう たべます。 (intent: already eaten)
✓ もう たべました。
"Already V-ed" needs past form ました; Vます is non-past.
✗ もう たべませんでした。 (intent: have already not eaten)
✓ もう たべました。 / まだ たべていません。
もう pairs with positive past (or ません = "no longer"). For "not yet" use まだ ていません.
✗ もう うちに かえります ました。
✓ もう うちに かえりました。
Past form is かえりました (drop ます, add ました). Do not stack ます+ました.
