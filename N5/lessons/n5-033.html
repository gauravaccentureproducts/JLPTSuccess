だけ

ENだけ - only / just (limiting particle)

JA「〜だけ」(げんてい)

Explanation

だけ means 'only' - limiting the scope to that one item. Used with affirmative or negative verbs (compare with しか, which forces negative).

Attaches to

noun, verb

Examples

JAコーヒーだけ 飲みます。
ENI drink only coffee.
JAわたしだけ 行きます。
ENOnly I will go.
JA見るだけです。
ENI'm only looking.
JA1,000円だけ あります。
ENI only have 1000 yen.
JAわたしだけ きょうしつに のこっています。
ENOnly I am still in the classroom.
JAコーヒーを 一ぱいだけ のみました。
ENI drank just one cup of coffee.
JAすこしだけ にほんごが わかります。
ENI understand only a little Japanese.
JAいちじかんだけ ねました。
ENI slept only one hour.
JAふたりだけ きました。
ENOnly two people came.
JAこれだけ かいました。
ENI bought only this much.

Common mistakes

✗ 一人だけです。
✓ 一人だけ来ました。
だけ is a particle that limits the preceding noun ('only N'); it is not a copula and does not pair with です in the same slot. Use it within a sentence: 'X だけ + verb' (only X did/has/is).
✗ コーヒーを だけ 飲みます。
✓ コーヒーだけ 飲みます。
だけ ('only') replaces を, not stacks after it. The pattern is NOUN + だけ + Verb (no を between). Adding を would double-mark the object; だけ already implies the noun is the object.
✗ コーヒーをだけ 飲みます。
✓ コーヒーだけ 飲みます。
だけ replaces を; the particles don't stack. NOUN + だけ + Verb is the canonical frame. Adding を before だけ doubles up an object-marking function that だけ already implies.

Wrong / corrected pairs

✗ いちにちだけにに べんきょうしました。
✓ いちにちだけ べんきょうしました。
だけ does not stack with に. Time-with-だけ takes no other particle.
✗ すこしだけしか たべません。
✓ すこししか たべません。 / すこしだけ たべます。
だけ and しか have similar meaning; do not stack. しか REQUIRES negative; だけ takes positive.
✗ ほんだけが あります。
✓ ほんだけ あります。
だけ replaces が/を for "only"; do not stack.
