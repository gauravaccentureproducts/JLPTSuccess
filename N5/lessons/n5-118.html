今（いま）/ すぐ / もう / まだ

ENいま / すぐ / もう / まだ - temporal adverbs

JA「いま／すぐ／もう／まだ」は 時間の すすみかたを いいます。「もう 食べました」「まだ 食べて いません」「すぐ 行きます」。

Explanation

Four core temporal adverbs. もう pairs with past (already done). まだ + negative = 'not yet', まだ + affirmative = 'still'.

Attaches to

adverbial

Examples

JAいま 行きます。
ENI'm going now.
JAもう ばんごはんを たべました。
ENI've already had dinner.
JAまだ たべていません。
ENI haven't eaten yet.
JAまだ ここに います。
ENI'm still here.
JAもう しゅくだいを しましたか。
ENHave you done your homework yet?
JAいま テストが おわりました。
ENThe test just ended.
JAもう しゅくだいを ぜんぶ しました。
ENI have already finished all my homework.
JAもう しゅくだいを しました。
ENI've already done my homework.
JAまだ ねないで ください。
ENPlease don't sleep yet.
JAすぐ きます。
ENI'll come right away.

Common mistakes

✗ まだ 食べました。
✓ もう 食べました。
もう = "already" (with affirmative past); まだ = "still / not yet" (often with negative). Don't mix.
✗ もう 食べていません。
✓ まだ 食べていません。
For "haven't eaten yet", use まだ + negative-progressive (まだ食べていません), not もう + negative.
✗ いまに 行きます。
✓ いま 行きます。
いま ('now') appears bare, with no particle. に would imply 'AT now' — but いま is already a momentary adverb. Same bare-form rule applies to すぐ/もう/まだ. Pattern: [aspect adverb] + Verb.

Wrong / corrected pairs

✗ もう たべません。 (intent: have not eaten yet)
✓ まだ たべていません。
もう = "already" (positive); まだ = "yet" (with negative).
✗ すぐ きました。 (intent: I came right away)
✓ すぐ きました。 (already correct)
すぐ = "right away / soon"; can be used with past-tense for "I came immediately".
✗ まだ いきますか。 (intent: have you gone yet)
✓ もう いきましたか。
For "have you done X yet?" use もう + past-question; まだ pairs with negative.
