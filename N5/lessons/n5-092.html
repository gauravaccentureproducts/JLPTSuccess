〜に〜があります / います

EN〜に〜が あります／います - 'there is X at Y' (location-first existence)

JA「〜に〜が あります／います」は ある ばしょに 何かが ある と いう いみです。「へやに ねこが います」。

Explanation

When introducing what's at a location, lead with the LOCATION (に) and follow with the SUBJECT (が) + あります/います. The location is the topic; the subject is new info.

Attaches to

noun_location, noun_subject

Examples

JAつくえの 上に 本が あります。
ENThere is a book on the desk.
JAきょうしつに がくせいが います。
ENThere are students in the classroom.
JAへやに テレビが ありません。
ENThere is no TV in the room.
JAこうえんに 子どもが いますか。
ENAre there children in the park?
JAつくえの 上に ほんが あります。
ENThere is a book on the desk.
JAこうえんに こどもが います。
ENThere are children in the park.
JAいえに ねこが います。
ENThere is a cat in the house.
JAつくえに ほんが あります。
ENThere is a book on the desk.
JAうちに ねこが います。
ENThere is a cat at my house.
JAがっこうに せんせいが います。
ENThere is a teacher at school.

Common mistakes

✗ つくえの 上で 本が あります。
✓ つくえの 上に 本が あります。
Location of EXISTENCE takes に, not で. で is for actions.
✗ つくえの 上は 本が あります。
✓ つくえの 上に 本が あります。
Location of existence takes に (location-particle). The pattern is: [location] + に + [thing] + が + あります/います. は in the location slot would make it a topic-comment frame, changing the syntactic role.
✗ つくえの 上に 本に あります。
✓ つくえの 上に 本が あります。
Subject marker が introduces the thing-that-exists. に marks the LOCATION, not the existing thing. Pattern: [location]に + [thing]が + あります/います.

Wrong / corrected pairs

✗ へやで テーブルが あります。
✓ へやに テーブルが あります。
Existence-location uses に, not で.
✗ がっこうで ねこが います。
✓ がっこうに ねこが います。
Static existence "be at" takes に. で is for actions occurring at a place.
✗ つくえに ほんに あります。
✓ つくえに ほんが あります。
The thing existing is marked with が, not に. に marks the LOCATION only.
