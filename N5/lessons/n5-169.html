Verb-た + ことがある

ENVerb-た + ことが ある - have done before (experience)

JA「〜た ことが ある」で、けいけんを いいます。「日本に 行った ことが あります」「すしを 食べた ことが ありません」。

Explanation

Verb-PAST + ことが + ある = 'have done at some point'. Asks/talks about life experience.

Attaches to

verb_ta

Common mistakes

✗ 日本に 行く ことが あります。 (intent: "have been to Japan")
✓ 日本に 行った ことが あります。
Past experience requires the た-form before ことがある. Plain form 行く + ことがある means "sometimes go (habit)", a different meaning.
✗ 日本へ 行くことが あります。
✓ 日本へ 行ったことが あります。
〜たことがある = 'have the experience of doing X'. Uses V-PAST + ことがある. 行く (dict) → 行った (past plain) + ことがあります. The dict form 行くことがあります would mean 'sometimes I go' (habitual), not 'I have been'.
✗ 日本へ 行くことが あります。
✓ 日本へ 行ったことが あります。
〜たことがある = 'have experience of doing X'. Uses V-past + ことがある. 行く → 行った. Dict-form (行く) + ことがある would mean 'I sometimes go' (habit), not 'I have been'.

Wrong / corrected pairs

✗ にほんに いく ことがあります。 (intent: have been to Japan)
✓ にほんに いった ことがあります。
"Have V-ed before" needs PAST PLAIN + ことがある. Dictionary form changes meaning to "sometimes do".
✗ いったことが ありますね。 (about your own experience)
✓ いったことが あります。
ね about your own experience implies seeking listener confirmation, which is odd.
✗ いったの ことがあります。
✓ いったことがあります。
Vた directly + ことがある, no nominalizer の.
