どこか / どこも

ENSomewhere / nowhere

JA「question word + か / も」

Explanation

Pattern: question word + か = 'some-' (indefinite). + も + affirmative = 'every-'. + も + negative = 'no-'.

Examples

JAどこかへ いきたいです。
ENI want to go somewhere.
JAどこも こんで います。
ENEverywhere is crowded.
JAきょうは どこへも いきません。
ENToday I'm not going anywhere.
JAどこにも 行きません。
ENI won't go anywhere.
JAどこかへ 行きませんか。
ENWon't you go somewhere?
JAどこにも ありません。
ENIt's nowhere.
JAきょうは どこも こんでいます。
ENEverywhere is crowded today.
JAどこかへ いきましょうか。
ENShall we go somewhere?
JAどこにも いきませんでした。
ENI didn't go anywhere.
JAどこかで みた ことが あります。
ENI've seen it somewhere.

Common mistakes

✗ どこか 行きません。
✓ どこも 行きません。
With negative, どこも = "(go) nowhere". どこか + negative is ungrammatical.
✗ どこも 行きたいです。
✓ どこか 行きたいです。
"Want to go somewhere" is affirmative wish - use どこか.
✗ どこかが いきたいです。
✓ どこかへ いきたいです。
Directional motion uses へ (or に). が would put 'somewhere' in subject position — doesn't fit 行きたい (want to go). The pattern is どこか + へ/に + Verb-stem + たい.

Wrong / corrected pairs

✗ どこも いきません。 (intent: I went somewhere)
✓ どこか いきました。
どこか = somewhere (positive); どこも = nowhere (with negative).
✗ どこかへ いきません か。 (intent: shall we go somewhere)
✓ どこかへ いきませんか。 (already correct)
どこか + へ for "somewhere" (direction); negative-question for invitation.
✗ どこにも いきました。
✓ どこにも いきませんでした。
どこにも requires negative ("nowhere" went).
