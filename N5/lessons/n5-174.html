〜なくてはならない

ENMust do (formal variant - same as いけない)

JA「〜なくては ならない」で、しなければ ならない こと、と いいます。「行か なくては なりません」。

Explanation

Negative て-form + は + いけない = 'must do' (lit. 'not doing won't work'). Casual contraction: 〜なくちゃ.

Examples

JAしゅくだいを しなくては ならない。
ENI must do my homework.
JAあした はやく おきなくては なりません。
ENI have to get up early tomorrow.
JAまいにち べんきょうしなくては ならない。
ENI have to study every day.
JAもっと べんきょうしなくては なりません。
ENI must study more.
JAあした はやく おきなくては なりませんか。
ENDo I have to wake up early tomorrow?
JAじかんを まもらなくては なりません。
ENYou must keep to the schedule.
JAあさごはんを たべなくては なりません。
ENI have to eat breakfast.
JAあさ はやく おきなくては なりません。
ENI must wake up early in the morning.
JAやくそくを まもらなくては なりません。
ENI must keep my promise.
JAはやく かえらなくては なりません。
ENI must return early.

Common mistakes

✗ 行かないと ならない。
✓ 行かなくては ならない。 / 行かないと いけない。
〜と pairs with いけない, not ならない. Mismatch: 〜なくては + ならない/いけない (both OK); 〜ないと + いけない (only).
✗ しゅくだいを しなくは ならない。
✓ しゅくだいを しなくては ならない。
Obligation: V-ない (drop い + くて) + は + ならない. The form しなくは skips the て-step. Full derivation: する → しない → しなくて → しなくては + ならない.
✗ しゅくだいを しないと ならない。
✓ しゅくだいを しなくては ならない。
Obligation 〜なくてはならない: V-ない + くて + は + ならない. Don't substitute と for くては — those are different constructions (n5-175 uses と + いけない).

Wrong / corrected pairs

✗ たべなくては だめです。
✓ たべなくては なりません。
Standard formal closer is なりません. だめ is more casual.
✗ たべないて はならない。
✓ たべなくては ならない。
Negative is なくて, not ないて.
✗ たべなくては なります。
✓ たべなくては なりません。
なる + ません (negative) = obligation. Positive なります changes meaning.
