Noun + の + Noun

ENNoun の Noun (covered by 〜の)

JA「い-Adj + めいし」で、けいよう-けいが めいしを せつめい します。「あかい 車」「大きい いえ」。

Explanation

の links nouns. Can mean possession (わたしの = 'my'), classification (日本語の = 'Japanese (language) X'), or general modification.

Examples

JAこれは わたしの 本です。
ENThis is my book.
JA日本語の 先生です。
ENI'm a teacher of Japanese.
JAあれは だれの かばんですか。
ENWhose bag is that?
JA友だちの お母さんの 名前は 田中さんです。
ENMy friend's mother's name is Tanaka.
JAにほんごの 先生は しんせつです。
ENThe Japanese teacher is kind.
JAともだちの くるまは あかいです。
ENMy friend's car is red.
JAわたしの いえの ちかくに こうえんが あります。
ENThere's a park near my house.
JAにほんの ほんが すきです。
ENI like Japanese books.
JAやまださんの くるまは あれです。
ENYamada-san's car is that one.
JAきょうの しんぶんを よみました。
ENI read today's newspaper.

Common mistakes

✗ 日本 学校
✓ 日本の 学校
Two nouns require の between them in modifier-modifier-of-noun position. Direct-juxtaposed compounds (日本人, 日本語) are exceptions where の is dropped by convention.
✗ これは わたしと 本です。
✓ これは わたしの 本です。
Possession between nouns: A + の + B = 'B of A' / 'A's B'. と would mean 'A AND B' (apposition or list). For 'my book', use わたし + の + 本.
✗ これは わたしと 本です。
✓ これは わたしの 本です。
Possession between two nouns uses の. わたし + の + 本 = 'my book'. と would mean 'I and book' (coordination), not 'my book'. Use の for the genitive/possessive link.

Wrong / corrected pairs

✗ にほん ほん。
✓ にほんの ほん。
Noun-noun linkage requires の.
✗ にほんの の ほん。
✓ にほんの ほん。
Single の; double is a typo.
✗ がっこうな ほん。
✓ がっこうの ほん。
Noun + noun uses の (not な; な is for na-Adj).
