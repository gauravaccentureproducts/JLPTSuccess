Verb-ていません

ENVerb-ていません - currently NOT doing / NOT in state

JA「Verb-て いません」は 「Verb-て います」の はんたいです。まだ して いない、と いう いみです。

Explanation

Negative form of ています. Two readings: (1) 'not currently doing' — 今 食べていません = 'I'm not eating now'; (2) 'not in resulting state / haven't done yet' — まだ 食べていません = 'I haven't eaten yet'. Often pairs with まだ for the latter.

Attaches to

verb_te

Common mistakes

✗ しっていません ("I don't know")
✓ しりません
しる is the famous exception. Affirmative しっています but negative しりません (NOT しっていません).
✗ まだ あさごはんを たべません。
✓ まだ あさごはんを たべていません。
'Not yet [verb]' uses まだ + ていません, NOT まだ + ません. The plain negative たべません means 'will not eat / does not eat'. The 'not yet' nuance requires the progressive negative ていません — まだ食べていません = I haven't eaten yet (as of now).
✗ まだ あさごはんを たべるていません。
✓ まだ あさごはんを たべていません。
Group 2 te-form drops る before て (たべる → たべて). Don't keep る. The progressive negative is V-te + います → ません. Pattern: V-te (dropped る) + いません.
✗ まだ あさごはん たべていません。
✓ まだ あさごはんを たべていません。
Object marker を required before たべる (transitive verb), even in the まだ + ていません frame. The 'not yet' aspect doesn't override transitivity.

Wrong / corrected pairs

✗ まだ たべません。
✓ まだ たべていません。
"Have not V-ed yet" requires Vていません, not Vません. Vません is future/habitual negative.
✗ まだ きませんでした。
✓ まだ きていません。
"Has not come yet" uses まだ + ていません, not past-negative ませんでした.
✗ もう たべていません。 (intent: already eaten)
✓ もう たべました。
もう pairs with positive past for "already V-ed". まだ + ていません = "not yet".
