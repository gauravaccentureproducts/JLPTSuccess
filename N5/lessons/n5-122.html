それから

ENそれから - 'and then / after that'

JA「それから」は 「その あと」と いう いみです。前の ことの あとに、つぎに する ことを いいます。

Explanation

それから begins a clause to mark sequence — 'after that, next'. Also carries an additive sense — 'and also / in addition' — when listing items: りんごと、それから、みかんも = 'apples, and also oranges'.

Attaches to

clause_start

Common mistakes

✗ シャワーを あびました。それから ねます。
✓ シャワーを あびました。それから ねました。
それから connects sequential past events; both clauses should match in tense.
✗ シャワーを あびました。そして、ねました。
✓ シャワーを あびました。それから、ねました。
それから = EXPLICIT TEMPORAL ('after that, then'). そして is neutral conjunction. For 'shower, then sleep' (clear time sequence), use それから. そして would feel softer/list-like.
✗ シャワーを あびて、それから ねました。
✓ シャワーを あびました。それから、ねました。
それから needs a complete sentence before it. Joining via te-form (あびて) makes both clauses one sentence, which doesn't need それから. For two-sentence sequence, use full polite past + period + それから.

Wrong / corrected pairs

✗ たべました。それからに、いきました。
✓ たべました。それから、いきました。
それから is itself a connector; do not add に or は after it.
✗ たべて、それから ねます。
✓ たべてから、ねます。 / たべて、それから ねます。
てから is tighter "right after"; それから is "after that, additionally". Subtle but distinct.
✗ それから、しんぶんを よみました それから、ねました。
✓ しんぶんを よんでから、ねました。
Repeating それから is choppy; consolidate with てから or chain with て-form.
