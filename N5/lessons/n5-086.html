な-Adjective negative: 〜じゃありません / 〜ではありません

ENな-Adjective negative - polite present

JAな-けいようしの ひてい

Explanation

な-adjectives form the negative the same way nouns do: + じゃありません (casual-polite) or + ではありません (formal). Just like the noun copula.

Attaches to

na_adjective

Common mistakes

✗ しずかくないです。
✓ しずかじゃありません。
な-adjectives DON'T use くない (that's い-adj). Use じゃありません, same as nouns.
✗ この へやは しずかくない です。
✓ この へやは しずかじゃありません。
な-adjective negative: stem + じゃありません (or でわありません/じゃない). The form しずかくない wrongly applies い-adj negation. な-adj uses じゃ/では, not くない.
✗ この へやは しずかくないです。
✓ この へやは しずかじゃありません。
な-adjective negative uses じゃありません (or でわありません/じゃない). The form くない is for い-adjectives only. Don't conflate the two adjective classes.

Wrong / corrected pairs

✗ しずかくないです。
✓ しずかじゃ ありません。 / しずかじゃないです。
na-adj negation uses じゃ; くない is for い-adjectives only.
✗ しずかくないです。
✓ しずかじゃないです。
na-Adj negative uses じゃない / ではない (not くない, which is for い-Adj).
✗ すきくない。
✓ すきじゃない。
すき is na-Adj despite ending in -き. Use じゃない / ではない.
