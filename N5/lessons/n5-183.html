Question word + か / も compounds

ENQuestion word + か / も - 'something / anyone' compounds

JA「だれか／なにか／どこか」は ある人・もの・ばしょを いいます。「だれも／なにも／どこも」は (with negative) いない・ない、と いう いみに なります。「だれかが 来ました」「なにも 食べませんでした」。

Explanation

Pattern: question word + か = 'some-' (indefinite). + も + affirmative = 'every-'. + も + negative = 'no-'.

Attaches to

question_word

Examples

JAなにか たべたいです。
ENI want to eat something.
JAなにも たべません。
ENI won't eat anything.
JAだれか 手つだってください。
ENSomeone please help me.
JAなにか 食べたいです。
ENI want to eat something.
JAなにか 飲みますか。
ENWill you drink something?
JAだれも 来ませんでした。
ENNobody came.
JAどこかへ 出かけますか。
ENAre you going somewhere?
JAなにか たべましたか。
ENDid you eat anything?
JAだれも きませんでした。
ENNo one came.
JAどこかへ いきましたか。
ENDid you go anywhere?

Common mistakes

✗ なに か 食べません。
✓ なにも 食べません。
With negative verbs, use なにも (nothing). なにか + negative is ungrammatical.
✗ なにを たべたいですか。
✓ なにか たべたいです。
なに + か = INDEFINITE pronoun ('something'). なに alone (with を) is a question ('what?'). For 'I want to eat SOMETHING' (vague desire, no specific item in mind), use なにか + たい. The Q-word + か pattern transforms the meaning.
✗ なにも たべたいです。
✓ なにか たべたいです。
Q-word + か = INDEFINITE 'something'. Q-word + も = inclusive 'nothing/anything' (requires negative). For 'I want to eat something' (positive desire), use なにか. なにも + positive predicate is grammatically wrong.

Wrong / corrected pairs

✗ なにも たべました。 (intent: I ate something)
✓ なにか たべました。
なにか = something (positive); なにも = nothing (with negative).
✗ なにも たべませんでしたか。 (intent: did you eat anything)
✓ なにか たべましたか。
なにか for "anything" in question; なにも + negative = "nothing".
✗ だれかが きました か。
✓ だれか きましたか。
だれか takes no particle (subject of intransitive いる/くる-type verbs).
