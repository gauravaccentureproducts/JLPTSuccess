Verb-stem + 〜かた

ENVerb-stem + 〜かた - way of doing

JA「Verb-stem + 〜かた」は 「やりかた」と いう いみです。「のみかた」「食べかた」「はなしかた」。

Explanation

Drop ます, add かた. 'How to / way of'. Pattern: Verb-stem + かた forms a noun. Examples: 読みかた (よみかた) = 'way of reading / how to read', 書きかた (かきかた) = 'how to write', 食べかた (たべかた) = 'how to eat'. Often appears in textbook titles and tutorials.

Attaches to

verb_stem_i

Examples

JAこの かんじの 読みかたは なんですか。
ENHow do you read this kanji?
JA日本語の 書きかたを おしえてください。
ENPlease teach me how to write Japanese.
JAえきまでの 行きかたが わかりません。
ENI don't know how to get to the station.

ENPlease teach me how to read Japanese.

Common mistakes

✗ 読むかた
✓ 読みかた
〜かた attaches to the verb-STEM (ます-form base), not the dictionary form. 読む → 読み + かた.
✗ 食べるかた
✓ 食べかた
For Group-2 verbs, drop the final る before adding かた: 食べる → 食べ + かた.
✗ この かんじの 読むかたは なんですか。
✓ この かんじの 読みかたは なんですか。
〜かた attaches to verb STEM (drop ます from ます-form). 読む → 読みます → 読み + かた = 読みかた. The form 読むかた wrongly uses dictionary form. Pattern: V-stem + かた = 'way of doing'.

Wrong / corrected pairs

✗ たべるかた。
✓ たべかた。
かた attaches to V-stem (drop る from G2): たべる→たべ+かた.
✗ よみますかた。
✓ よみかた。
Drop ます from G1 stem: よみます→よみ+かた.
✗ たべかたを おしえます。 (intent: for the dish, not the action)
✓ つくりかたを おしえます。
For "way to make" use つくりかた; たべかた = "way to eat".
