い-Adjective past: 〜かったです

ENい-Adjective past - was [adjective]

JAい-けいようしの かこ

Explanation

Drop the final い and add かった for the past form. Add です for polite. NEVER use でした with い-adjectives - that's only for nouns and な-adjectives.

Attaches to

i_adjective

Examples

JAきのうは あつかったです。
ENYesterday was hot.
JAその えいがは おもしろかったです。
ENThat movie was interesting.
JAテストは むずかしかったですか。
ENWas the test difficult?
JAきのうの ばんごはんは おいしかったです。
ENYesterday's dinner was delicious.
JAきのうは あつかったですか。
ENWas it hot yesterday?
JAあの えいがは よかったですか。
ENWas that movie good?
JAあの えいがは ながかったです。
ENThat movie was long.
JAテストは むずかしかったです。
ENThe test was difficult.
JAえいがは おもしろかったです。
ENThe movie was interesting.
JAりょこうは たのしかったです。
ENThe trip was fun.

Common mistakes

✗ たかいでした。
✓ たかかったです。
い-adjectives DON'T use でした. The past is built by changing い → かった.
✗ いいかった。
✓ よかった。
いい (good) → よかった (was good). The whole word changes to よ-stem.
✗ きのうは あついでした。
✓ きのうは あつかったです。
い-adjective past: drop い, add かった, then です. あつい → あつ + かった + です = あつかったです. The form あついでした (い-adj + でした) is wrong — でした is for nouns/な-adj only.

Wrong / corrected pairs

✗ 高いでした。
✓ 高かったです。
い-adjective past is い→かった + です (NOT い + でした).
✗ たかいかったです。
✓ たかかったです。
Drop final い then add かった: 高い→高かった (not 高いかった).
✗ いいかったです。
✓ よかったです。
いい uses irregular stem よ for past: よかった.
