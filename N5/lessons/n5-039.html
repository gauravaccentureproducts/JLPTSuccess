これ / それ / あれ / どれ

ENPronouns: this / that / which one

JA「これ・それ・あれ・どれ」(ぶつ)

Explanation

The こ/そ/あ/ど pronouns refer to OBJECTS (or abstract things). これ = near speaker, それ = near listener, あれ = far from both, どれ = which one (question). They stand alone - DON'T put a noun after them. For 'this book' use この (n5-015), not これ.

Examples

JAこれは ほんです。
ENThis is a book.
JAそれは なんですか。
ENWhat is that (near you)?
JAあれは わたしの くるまじゃありません。
ENThat isn't my car.
JAどれが いちばん すきですか。
ENWhich one do you like best?
JAこれを ください。
ENPlease give me this.
JAこれは 何ですか。
ENWhat is this?
JAそれは わたしの ほんです。
ENThat's my book.
JAこれは わたしの ペンです。
ENThis is my pen.
JAそれは あなたの ほんですか。
ENIs that your book?
JAあれは なんですか。
ENWhat is that over there?

Common mistakes

✗ これは本ですか。それは机です。
✓ これは本です。それは机です。
これ/それ/あれ are full noun-pronouns standing alone as the subject. Beginners mix them with この/その/あの (which are NOUN MODIFIERS and require a following noun). これ本 is ungrammatical; この本 is correct.
✗ あれは ほんです。
✓ これは ほんです。
Same kosoado proximity rule as n5-014. これ for near-speaker, それ for near-listener, あれ for far-from-both. Identifying an object the speaker is showing uses これ.
✗ あれは ほんです。 (when speaker is holding the book)
✓ これは ほんです。
Proximity rule: これ for things near the speaker (in hand, nearby). あれ would put the book far from both speaker and listener — wrong if speaker is presenting the item. Choose by physical proximity, not by what 'sounds polite'.

Wrong / corrected pairs

✗ これ ほん。
✓ これは ほんです。
Pronoun これ requires は + complete sentence with です.
✗ これ ほんは わたしの。
✓ この ほんは わたしの です。
Before noun use この (determiner); これ is standalone pronoun.
✗ どれは いいですか。
✓ どれが いいですか。
どれ as question takes が, never は.
