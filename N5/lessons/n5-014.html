これ／それ／あれ／どれ

ENPronouns: this / that-near-listener / that-far / which-one (standalone, no noun follows)

JA「これ・それ・あれ・どれ」は もの や こと を しめす ことばです。「これ」(はなす人の ちかく)、「それ」(きく人の ちかく)、「あれ」(どちらも とおい)、「どれ」(しつもん)。

Explanation

The こ/そ/あ/ど pronouns refer to OBJECTS (or abstract things). これ = near speaker, それ = near listener, あれ = far from both, どれ = which one (question). They stand alone - DON'T put a noun after them. For 'this book' use この (n5-015), not これ.

Attaches to

standalone

Examples

JAこれは ほんです。
ENThis is a book.
JAそれは なんですか。
ENWhat is that (near you)?
JAあれは くるまじゃありません。
ENThat (over there) is not a car.
JAどれが いちばん すきですか。
ENWhich one do you like best?
JAこれを ください。
ENPlease give me this.
JAそれは わたしの かさです。
ENThat is my umbrella.
JAこれは 父の しゃしんです。
ENThis is my father's photograph.
JAこれは ペンです。
ENThis is a pen.
JAそれは くるまですか。
ENIs that a car?
JAあれは わたしの いえです。
ENThat over there is my house.

Common mistakes

✗ これ ほんは おもしろいです。
✓ この ほんは おもしろいです。 / これは おもしろい ほんです。
これ stands alone - never followed by a noun. For 'this book' use この + noun (see n5-015).
✗ あれは どれですか。
✓ あれは なんですか。
どれ asks 'which one (of these options)?' - it's a chooser. To ask 'what is that?' use なん/なに.
✗ それは ほんです。
✓ これは ほんです。
Choose between これ/それ/あれ based on speaker-listener proximity. これ = near speaker (in their hand or nearby). それ = near listener. あれ = far from both. A speaker showing a book they're holding uses これ.

Wrong / corrected pairs

✗ これ ペンですか。
✓ これは ペンですか。
これ is a bare pronoun; needs は to be the topic of a question.
✗ これ ほんは わたしの です。
✓ この ほんは わたしの です。
これ stands alone (this); この + N modifies a noun (this book). Cannot use これ before a noun.
✗ どれは いいですか。 (intent: which is good)
✓ どれが いいですか。
Question word どれ takes が, never は. (は cannot topicalize unknown info.)
