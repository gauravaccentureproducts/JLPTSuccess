など

ENなど - etc. (and so on)

JA「〜など」は たくさんの 中から いくつかを れいに あげます。「りんご・みかん など」。

Explanation

など follows a list (often with や) to add 'etc., and so on'. More formal/written counterpart of casual とか. Use と for a closed exhaustive list ('A and B'), や〜など for an open representative list ('A, B, etc.').

Attaches to

noun

Examples

JAりんごや バナナなどを よく 食べます。
ENI often eat apples, bananas, etc.
JAペンや ノートなどを 買いました。
ENI bought pens, notebooks, and so on.
JAそふぼや ともだちなどに あいました。
ENI met my grandparents, friends, and others.
JAすしや てんぷらなどの 日本りょうりが すきです。
ENI like Japanese food such as sushi and tempura.
JAペンや ノートなどを かいました。
ENI bought pens, notebooks, and so on.
JAりんごや バナナなどを たべました。
ENI ate apples, bananas, and so on.
JAりんごや みかんなどを 買いました。
ENI bought apples, oranges, and so on.
JAノートや ペンなどを かいました。
ENI bought things like notebooks and pens.
JAえいがや ほんなどが すきです。
ENI like things like movies and books.
JAバスや でんしゃなどで いきます。
ENI go by bus, train, etc.

Common mistakes

✗ りんご、みかん、バナナなど。
✓ りんごやみかんやバナナなど（が好きです）。
など typically pairs with や ('and similar things') to indicate a non-exhaustive list. Listing items with bare commas + など omits the connector や that makes the 'etc.' reading natural in Japanese.
✗ りんごや バナナや などを よく 食べます。
✓ りんごや バナナなどを よく 食べます。
など follows the LAST item in a や-list to mean 'and so on'. Don't repeat や before など. Pattern: A や B など (NO や immediately before など). Or use just A や B if 'and so on' is implied.
✗ りんごと バナナと よく 食べます。
✓ りんごや バナナなどを よく 食べます。
や (non-exhaustive list) + など ('and so on') signals 'apples, bananas, etc.' (illustrative examples). と would make the list EXHAUSTIVE (only these two). For 'such things as', use the や + など frame.

Wrong / corrected pairs

✗ コーヒーや こうちゃ などを のみました。
✓ コーヒーや こうちゃなどを のみました。
Spacing: など attaches directly to the last item without space.
✗ コーヒーと こうちゃ などを のみました。
✓ コーヒーや こうちゃなどを のみました。
と is exhaustive; pair など with や (non-exhaustive) for "X, Y, etc.".
✗ りんご など を かいました。
✓ りんごなどを かいました。
No space between item and など; など functions as suffix.
