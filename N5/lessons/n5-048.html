どこ

ENWhere (covered by ここ-series)

JAばしょの ことば

Explanation

Same こ/そ/あ/ど logic but for LOCATIONS. Take は, が, に, で, から, まで etc. as needed. ここ/そこ/あそこ are statements; どこ asks 'where?'. With も + negative: どこにも いきません = 'I'm going nowhere'.

Examples

JAここは としょかんです。
ENHere is the library.
JAそこに ねこが います。
ENThere is a cat there.
JAどこで かいましたか。
ENWhere did you buy it?
JAどこへ いきたいですか。
ENWhere do you want to go?
JAどこにも いきません。
ENI'm not going anywhere.
JAどこへ 行きましたか。
ENWhere did you go?
JAだれが きましたか。
ENWho came?
JAどこから きましたか。
ENWhere did you come from?
JAどこで まちましたか。
ENWhere did you wait?
JAどこに すんで いますか。
ENWhere do you live?

Common mistakes

✗ 銀行はどこにありますか。 → 銀行はどこですか。
✓ Both correct; どこにある focuses on existence/location, どこです focuses on identification.
Beginners conflate どこですか (where is it, asking for the place name) with どこにありますか (where does it exist, asking for directions). Use です for asking 'name of place'; use にある for asking 'how do I get there'.
✗ あそこは としょかんです。
✓ ここは としょかんです。
When introducing the location speaker is at, ここ (near speaker). For a place far from both, あそこ. The example shows speaker pointing out where they are — ここ.
✗ ここは としょかんですですか。
✓ ここは としょかんですか。
Single です + か is the standard question pattern. Don't double the copula. Pattern: TOPIC + は + NOUN + ですか. 'ですですか' is structurally wrong (double copula).

Wrong / corrected pairs

✗ どこに いきますか。 (intent: where do you live)
✓ どこに すんでいますか。
いきます = "go (motion)"; すんでいます = "live (residence)". Different verbs for different questions.
✗ どこから きましたか。 (asking origin politely)
✓ どちらから いらっしゃいましたか。
In formal contexts use どちら + irassyaru; どこ is neutral.
✗ どこに あそびますか。
✓ どこで あそびますか。
Action location takes で; に would mean motion-destination.
