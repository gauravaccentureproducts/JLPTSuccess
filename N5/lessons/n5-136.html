Adj + N (combined)

ENAdjective + noun (covered by い-adj+N and な-adj+N)

JAけいようし（い／な）＋ めいし。い-けいようしは そのまま、な-けいようしは 「な」を つけて めいしに つける。

Explanation

い-adjectives modify nouns directly, no particles needed. Keep the い: 高い 山 = 'tall mountain'.

Examples

JA高い 山が 見えます。
ENI can see a tall mountain.
JAおもしろい 本を 読みました。
ENI read an interesting book.
JAあつい コーヒーが すきです。
ENI like hot coffee.
JA大きい 白い いぬが すきです。
ENI like big white dogs.
JA大きい 本を 買いました。
ENI bought a big book.
JAおもしろい えいがを 見ました。
ENI watched an interesting movie.
JAやさしい 先生は たなかさんです。
ENThe kind teacher is Ms. Tanaka.
JAおおきい ねこが います。
ENThere is a big cat.
JAしずかな まちが すきです。
ENI like quiet towns.
JAたかい ほんを かいました。
ENI bought an expensive book.

Common mistakes

✗ 高いの本です。 / 静かの部屋です。
✓ 高い本です。 / 静かな部屋です。
い-adjectives attach directly to nouns (高い+本 = 高い本). な-adjectives need な (静か+な+部屋 = 静かな部屋). The big mistake is inserting の between adjective and noun (Mandarin 的 transfer).
✗ 高いの 山が 見えます。
✓ 高い 山が 見えます。
い-adjectives modify nouns DIRECTLY. 高い + 山 (no particle, no の). The form 高いの would nominalize 高い as 'the high one'. For 'a high mountain', use bare adjective + noun.
✗ 高いに 山が 見えます。
✓ 高い 山が 見えます。
i-adjective + Noun: direct attachment, no に, no の, no な. 高い + 山 forms a single noun phrase. Adding any particle between adjective and noun breaks the modification.

Wrong / corrected pairs

✗ たかいなほん。
✓ たかい ほん。
い-Adj + N directly (no な).
✗ しずか へや。
✓ しずかな へや。
na-Adj + N requires な linker.
✗ いいの ひと。
✓ いい ひと。
い-Adj attaches directly; do not insert の.
