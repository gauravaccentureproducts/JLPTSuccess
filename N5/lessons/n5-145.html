〜とおもいます

EN〜と おもいます - I think (that)

JA「〜と おもいます」で、じぶんの かんがえを いいます。「あした あめが ふると おもいます」。

Explanation

Embed a plain-form clause + と + おもいます to express opinion. と marks the quoted/embedded thought.

Attaches to

plain_clause

Examples

JAあした 雨が ふると おもいます。
ENI think it'll rain tomorrow.
JAいいと おもいます。
ENI think it's good.
JAこの えいがは おもしろいと おもいます。
ENI think this movie is interesting.
JAあした あめが ふると おもいます。
ENI think it will rain tomorrow.
JAやまださんは いい 先生だと おもいます。
ENI think Yamada-san is a good teacher.
JAあの ホテルは たかいと おもいます。
ENI think that hotel is expensive.
JAこの りょうりは おいしいと おもいます。
ENI think this dish is delicious.
JAあした、 あめが ふると おもいます。
ENI think it will rain tomorrow.
JAやまださんは げんきだと おもいます。
ENI think Yamada-san is well.
JAむずかしいと おもいます。
ENI think it is difficult.

Common mistakes

✗ 〜です と おもいます
✓ 〜だ と おもいます (or just 〜と おもいます)
Use PLAIN form before と: noun + だ, い-adj + plain. Don't insert です.
✗ あした 雨が ふるから おもいます。
✓ あした 雨が ふると おもいます。
Quoted-thought marker is と, not から. と marks the content of thinking/saying. から marks a reason. For 'I think [content]', use [clause] + と + おもう.
✗ あした 雨が ふるとが おもいます。
✓ あした 雨が ふると おもいます。
Quote-marker と attaches directly to the plain-form verb (no が between). Pattern: [clause-plain]と + おもう. Don't add a subject-marker after と — the と clause is the OBJECT of おもう.

Wrong / corrected pairs

✗ あめが ふりますと おもいます。
✓ あめが ふると おもいます。
〜とおもう takes plain form (Vる/Vない/Vた); polite ます-form before と is non-standard.
✗ おもいます、あめが ふる と。
✓ あめが ふると おもいます。
Quoted clause + と + おもいます = canonical order. Cannot post-pose the と-clause.
✗ がくせいだと おもう です。
✓ がくせいだと おもいます。
おもう can be plain; if attaching to something stack-able, use polite おもいます (consistent register).
