何（なに／なん）

ENWhat - pronounced なに or なん depending on what follows

JA「何（なに／なん）」は きく ときの ことばです。よみかたは ふたつ あります。「なに」と「なん」、どちらも かんじは「何」です。「なに」は ものを きく とき。「なん」は かずや「です」の まえで。

Explanation

なに and なん both mean 'what'. The pronunciation depends on what follows: なん before です, ですか, でしょう, and counters (なんじ, なんにん, なんさい); なに before を, が, も, は. Memorize the pattern. なんで means 'how/why' (informal); なぜ is the formal 'why'.

Common mistakes

✗ なんを たべますか。
✓ なにを たべますか。
なに before を (and が, も). Never なん before を. The trigger for なん is です / ですか / でしょう / counters.
✗ なにじですか。
✓ なんじですか。
Counters always trigger なん: なんじ, なんにん, なんさい, なんがつ, なんにち, なんようび.
✗ なにが たべますか。
✓ なにを たべますか。
なに is the direct OBJECT of たべる (eat), so it takes を, not が. が would put なに in the subject slot (asking 'what eats?' — nonsense). The direct-object rule: transitive verb + を + object.

Wrong / corrected pairs

✗ なにじですか。
✓ なんじですか。
Before counters (じ, にん, がつ, etc.), 何 is read なん. なに is used standalone or before non-counter nouns.
✗ なにを たべます？ (in writing, end with ?)
✓ なにを たべますか。
Formal Japanese ends questions with か。, not ?. The question particle is the marker.
✗ なんが すきですか。
✓ なにが すきですか。
Before bare が (no counter), use なに. Both readings exist; the rule is なん before counters / なに elsewhere.
