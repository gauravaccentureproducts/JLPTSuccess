Verb-た + ほうがいい

ENVerb-た + ほうが いい - should do

JA「〜た ほうが いい」で、アドバイスを します。「もっと 休んだ ほうが いい」「はやく ねた ほうが いいです」。

Explanation

Verb-PAST + ほうが + いい = 'you'd better / should do'. Use PAST form even for future advice.

Attaches to

verb_ta

Examples

JAもう ねたほうが いいですよ。
ENYou'd better go to bed already.
JAびょういんに 行ったほうが いいです。
ENYou should go to the hospital.
JAはやく ねたほうが いいです。
ENIt's better to go to bed early.
JAはやく ねた ほうが いいですよ。
ENYou should go to sleep early.
JAやすんだ ほうが いいです。
ENYou should rest.
JAさんぽした ほうが いいですよ。
ENYou should go for a walk.
JAくすりを のんだ ほうが いいです。
ENYou should take medicine.
JAやさいを たべた ほうが いいですよ。
ENYou should eat vegetables.
JAやさいを たべた ほうが いいです。
ENYou should eat vegetables.
JAびょういんに いった ほうが いいです。
ENYou should go to the hospital.

Common mistakes

✗ 行く ほうが いいです。
✓ 行った ほうが いいです。
Recommendations use the た-form: 〜たほうがいい. Plain form 〜ほうがいい is grammatical but sounds like a comparison ("the going option is better"), not advice.
✗ もう 寝るほうが いいですよ。
✓ もう 寝たほうが いいですよ。
〜たほうがいい = 'should do X / better to do X' takes V-PAST plain, not dict. 寝る → 寝た + ほうがいい. Even though the meaning is future-oriented advice, the form requires past. Pattern: V-た + ほうがいい.
✗ もう 寝るほうが いいですよ。
✓ もう 寝たほうが いいですよ。
〜たほうがいい = 'should do X / better to do X'. Uses V-PAST plain + ほうがいい. 寝る → 寝た. Even though the advice is future-oriented, the form is past.

Wrong / corrected pairs

✗ たべる ほうがいい。 (intent: should eat)
✓ たべた ほうがいい。
Advisory "should V" in N5 uses Vた (past plain) + ほうがいい. Vる makes it a generic comparison.
✗ たべない ほうがいい です ね。 (commanding)
✓ たべない ほうがいいですよ。
Advisory uses よ (informing) more than ね (seeking agreement).
✗ たべた ほうが いい だ。
✓ たべた ほうが いい。
いい is an adjective; do not append だ. Use plain いい or polite いいです.
