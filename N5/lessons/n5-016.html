ここ／そこ／あそこ／どこ

ENLocation words: here / there-near-listener / over-there / where

JAばしょの ことば

Explanation

Same こ/そ/あ/ど logic but for LOCATIONS. Take は, が, に, で, から, まで etc. as needed. ここ/そこ/あそこ are statements; どこ asks 'where?'. With も + negative: どこにも いきません = 'I'm going nowhere'.

Attaches to

standalone

Examples

JAここは としょかんです。
ENHere is the library.
JAそこに ねこが います。
ENThere is a cat there.
JAどこで かいましたか。
ENWhere did you buy it?
JAあそこは しずかです。
ENOver there is quiet.
JAどこにも いきません。
ENI'm not going anywhere.
JAここで まちます。
ENI will wait here.
JAホテルは あそこです。
ENThe hotel is over there.
JAここは わたしの きょうしつです。
ENThis (here) is my classroom.
JAそこに ペンが あります。
ENThere (near you) is a pen.
JAあそこに ぎんこうが あります。
ENOver there is a bank.

Common mistakes

✗ ここに ねこは あります。
✓ ここに ねこが います。
Use います for animate things (people, animals); あります is only for inanimate things.
✗ どこも しずかです。 (with affirmative - meaning 'every place is quiet')
✓ どこも しずかです。 IS valid - but watch the rule: どこも + AFFIRMATIVE = 'everywhere'; どこも/どこにも + NEGATIVE = 'nowhere'.
Polarity flip: question word + も + affirmative = 'every', + negative = 'no'. Same rule applies to だれも, なにも, いつも.
✗ あそこは としょかんです。
✓ ここは としょかんです。
ここ marks the SPEAKER'S CURRENT LOCATION ('here, where I am'). あそこ would be 'over there, far from us'. For introducing the speaker's surrounding building, use ここ.

Wrong / corrected pairs

✗ ここ は ぎんこうですか。
✓ ここは ぎんこうですか。
Spacing: は attaches directly to ここ; do not insert a space.
✗ どこは ぎんこうですか。
✓ ぎんこうは どこですか。
Question word どこ takes が — but typical phrasing is "X は どこ ですか" (where is X), not "どこは X ですか".
✗ こちらは どこ ですか。 (asking your own location)
✓ ここは どこ ですか。
こちら is polite/direction ("this way / over here"); ここ is the spatial pronoun ("here").
