Verb-ました

ENPolite past affirmative verb ending

JAていねいな かこけい

Explanation

ました replaces ます to form the polite past affirmative.

Attaches to

verb_stem

Examples

JAきのう えいがを みました。
ENI watched a movie yesterday.
JAあさごはんを たべました。
ENI ate breakfast.
JAなにを かいましたか。
ENWhat did you buy?
JAともだちに あいました。
ENI met a friend.
JAだれが きましたか。
ENWho came?
JA父に とけいを もらいました。
ENI received a watch from my father.
JA母と えいがを 見ました。
ENI watched a movie with my mother.
JAきのう、えいがを みました。
ENI watched a movie yesterday.
JAけさ、しんぶんを よみました。
ENI read the newspaper this morning.
JAもう ひるごはんを たべました。
ENI've already eaten lunch.

Common mistakes

✗ きのう たべますでした。
✓ きのう たべました。
ました replaces ます - don't add でした to ます. でした is only for です.
✗ きのう えいがを みるました。
✓ きのう えいがを みました。
Past polite: drop ます, add ました. みる (Group 2) → みます → みました. The form みるました doesn't exist — you can't keep dictionary る AND have ました. Pattern: V-stem + ました.
✗ きのう えいがを みるた。
✓ きのう えいがを みました。
Polite past: drop ます from V-ます-form, add ました. みる (dict) → みます → みました. Don't keep dict-form る with past marker. Pattern: V-stem + ました.

Wrong / corrected pairs

✗ たべるました。
✓ たべました。
Group-2: drop る then add ました. たべる→たべ+ました.
✗ いきました でした。
✓ いきました。
ました already encodes past + polite. Do not add でした (which is past of です/copula).
✗ きのうは えいがを みます ました。
✓ きのうは えいがを みました。
Past form is みました (drop ます, add ました). Do not stack ます + ました.
