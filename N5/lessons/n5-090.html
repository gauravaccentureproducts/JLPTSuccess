あります

ENあります - there is / exists (inanimate things)

JA「あります」(ものの そんざい)

Explanation

あります expresses existence for INANIMATE things (objects, plants, abstract things, events). Subject is marked with が; location with に. For animate (people, animals), use います instead.

Attaches to

noun_subject

Examples

JAつくえの 上に ほんが あります。
ENThere is a book on the desk.
JAへやに テレビが ありません。
ENThere is no TV in the room.
JAきのう しけんが ありました。
ENThere was an exam yesterday.
JAなにか しつもんが ありますか。
ENDo you have any questions?
JAあそこに なにが ありますか。
ENWhat is over there?
JAきのう ここに ペンが ありました。
ENThere was a pen here yesterday.
JAあした パーティーが あります。
ENThere is a party tomorrow.
JAつくえに ほんが あります。
ENThere's a book on the desk.
JAきょう、テストが あります。
ENThere's a test today.
JAおかねが あまり ありません。
ENI don't have much money.

Common mistakes

✗ へやに ねこが あります。
✓ へやに ねこが います。
あります is for inanimate things only. Animals and people use います.
✗ テレビが あるです。
✓ テレビが あります。
あります is already the polite form. Don't add です.
✗ つくえの 上に ほんが います。
✓ つくえの 上に ほんが あります。
ほん (book) is INANIMATE — uses あります. います is for animate things (people, animals). Memorize: ある/あります for objects, events, abstract concepts; いる/います for living beings.

Wrong / corrected pairs

✗ 人が あります。
✓ 人が います。
います for animate; あります for inanimate. Plants take あります.
✗ ねこが あります。
✓ ねこが います。
あります is for inanimate / non-living; animate beings (people, animals) use います.
✗ いすに ほんが います。
✓ いすに ほんが あります。
Books are inanimate → あります. います is for living things.
