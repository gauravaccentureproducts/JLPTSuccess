どれ / どの / どちら

ENどれ / どの / どちら - 'which' (selection from options)

JA「どれ・どの・どちら」は えらぶ ときに つかう ことばです。「どれが いいですか」「どちらが すきですか」。

Explanation

Three 'which' forms: どれ stands alone for 3+ choices. どの precedes a noun. どちら is the polite form, also used for 2-way comparison.

Common mistakes

✗ どれ 本
✓ どの 本
Before a noun, the form is どの (NOUN-modifier): どの 本 = 'which book'. Standalone どれ is for pronoun use: どれが あなたの = 'which one is yours'. Different syntactic roles, different forms.
✗ どれは あなたの ですか。
✓ どれが あなたの ですか。
Question word どれ takes が, not は. 'Which one is yours?' — どれ is the new-information subject. The answer is これが私の (with が). The は-form would be ungrammatical at N5.
✗ どれは あなたの ですか。
✓ どれが あなたの ですか。
どれ is a question word, so the subject particle is が (not は). Same rule as だれが/なにが — question words live in the new-information slot, which が marks.

Wrong / corrected pairs

✗ どの は いいですか。
✓ どれが いいですか。
どの requires a noun; for pronoun-only "which" use どれ + が.
✗ どちらは いいですか。 (about 3 options)
✓ どれが いいですか。
どちら is for binary choice (between 2); どれ for 3+.
✗ どれは ペンですか。
✓ どれが ペンですか。
Q-word どれ takes が.
