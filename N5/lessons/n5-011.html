や

ENAnd - non-exhaustive listing (these and others)

JA「や」は いくつかの ものを ならべて いう ことばです。ぜんぶ では ありません。れいを あげる ときに つかいます。「りんごや みかんを 買います」。

Explanation

や lists nouns NON-EXHAUSTIVELY - A, B, and others (implied). Often paired with など (etc.) at the end of the list. Contrast with と, which is EXHAUSTIVE - only A and B and nothing else.

Attaches to

noun

Examples

JAりんごや バナナを たべます。
ENI eat apples, bananas, etc.
JAペンや ノートなどを かいました。
ENI bought pens, notebooks, and so on.
JAなにや なにを かいましたか。
ENWhat and what did you buy?
JAこうえんや えきの ちかくに カフェが あります。
ENThere are cafes near parks, stations, etc.
JAあさ パンや たまごを たべます。
ENIn the morning I eat bread, eggs, etc.
JAアメリカや 日本へ いきました。
ENI went to America, Japan, and other places.
JAきのう ぎゅうにくや さかなを たべました。
ENYesterday I ate beef, fish, and so on.
JAりんごや みかんを かいました。
ENI bought apples, oranges, etc.
JAねこや いぬが すきです。
ENI like cats and dogs (among others).
JAノートや えんぴつなどを かいました。
ENI bought notebooks, pencils, etc.

Common mistakes

✗ りんごや バナナと たべます。
✓ りんごや バナナを たべます。 (non-exhaustive) / りんごと バナナを たべます。 (exhaustive)
Don't mix や (non-exhaustive) and と (exhaustive) in the same listing. Pick one based on whether you want to imply 'and others'.
✗ りんご や あかい や...
✓ りんごや バナナや みかんなど
や connects nouns only - never adjectives or verbs. For chaining adjectives use the て-form.
✗ りんごと バナナを たべます。
✓ りんごや バナナを たべます。
や lists NON-EXHAUSTIVE examples ('apples, bananas, etc.'). と would make the list EXHAUSTIVE ('apples and bananas, period'). Use や when the listed items are illustrative samples; use と when the list is complete.

Wrong / corrected pairs

✗ コーヒー、こうちゃ、おちゃを のみます。 (intent: among others)
✓ コーヒーや こうちゃや おちゃを のみます。
や makes a non-exhaustive list. Just commas without や is not standard for "things like X, Y".
✗ コーヒーや こうちゃや のみます。
✓ コーヒーや こうちゃ などを のみます。
When や ends a list, add など or another connector before the verb. Bare や cannot directly attach to を.
✗ りんごや みかんと バナナを かいました。
✓ りんごや みかんや バナナを かいました。
Mix of や and と is wrong. と is exhaustive ("just A and B"); や is non-exhaustive ("A, B, etc.").
