いつか / いつも

ENSometime / always

JA「question word + か / も」

Explanation

Pattern: question word + か = 'some-' (indefinite). + も + affirmative = 'every-'. + も + negative = 'no-'.

Examples

JAいつか にほんへ いきたいです。
ENSomeday I want to go to Japan.
JAいつも 7時に おきます。
ENI always wake up at 7.
JAちちは いつも しごとで いそがしいです。
ENMy father is always busy with work.
JAいつか いっしょに ばんごはんを 食べましょう。
ENLet's have dinner together sometime.
JAいつも 七時に おきます。
ENI always wake up at seven.
JAいつか 日本へ 行きたいです。
ENSomeday I want to go to Japan.
JAいつか おかあさんに 会いたいです。
ENSomeday I want to meet your mother.
JAいつか にほんに いきたいです。
ENI want to go to Japan someday.
JAいつも コーヒーを のみます。
ENI always drink coffee.
JAいつか また あいましょう。
ENLet's meet again someday.

Common mistakes

✗ いつも 日本に 行きたいです。 (intent: "someday")
✓ いつか 日本に 行きたいです。
いつも = "always" (every time); いつか = "someday / sometime in the future". For aspirations, use いつか.
✗ いつか コーヒーを 飲みます。 (intent: "always")
✓ いつも コーヒーを 飲みます。
For habit "always drink coffee", use いつも. いつか implies a single future occasion.
✗ いつ にほんへ いきたいです。
✓ いつか にほんへ いきたいです。
いつ alone is a specific time-question ('when?'). いつか with か = INDEFINITE ('someday', vague future). For a non-specific aspirational time ('someday I want to go'), use いつか + に + へ + V-stem + たい.

Wrong / corrected pairs

✗ いつも にほんに いきたい です。 (intent: someday I want to go)
✓ いつか にほんに いきたいです。
いつか = someday (indefinite future); いつも = always (habit).
✗ いつも にほんへ いったり する。
✓ いつも にほんに いきます。 / いつも〜たり〜たり する。
For habit use いつも + simple Vます or 〜たり〜たり pattern.
✗ いつかに きます。
✓ いつか きます。
いつか takes no particle.
