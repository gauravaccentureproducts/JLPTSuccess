Verb-ない

ENPlain non-past negative

JAふつうの ひていけい

Explanation

ない is the plain (casual) negative. Group 1: change the final -u to -a + ない (のむ → のまない; かう → かわない - note わ for vowel-stems). Group 2: drop る and add ない. する → しない, くる → こない.

Attaches to

verb_root

Examples

JAおさけを のまない。
EN[I] don't drink alcohol.
JAにくを たべない。
EN[I] don't eat meat.
JAあした こない。
EN[I] won't come tomorrow.
JAべんきょうしない 人。
ENA person who doesn't study.
JAそんな ことは 言わないで ください。
ENPlease don't say such things.
JAにほんごを 話さない。
ENI don't speak Japanese.
JAお金が ないから、行かない。
ENI don't have money, so I won't go.
JAコーヒーを のまない。
ENI don't drink coffee.
JAあした、いかない。
ENI won't go tomorrow.
JAこれを たべない ほうが いい。
ENBetter not to eat this.

Common mistakes

✗ かうない
✓ かわない
For -う ending verbs, the negative stem changes う → わ (not あ). Same for あう → あわない, いう → いわない.
✗ おさけを のまる ない。
✓ おさけを のまない。
Group-1 plain negative: drop final う-row, add あ-row + ない. のむ → のま + ない = のまない. Don't insert る — that's only for Group-2 dictionary form. Pattern: V-stem (with あ-row vowel) + ない.
✗ おさけを のまるない。
✓ おさけを のまない。
Group 1 plain negative: change last う-row to あ-row, add ない. のむ → のま + ない = のまない. Don't insert る — that's for Group 2 only. Pattern: stem (あ-row) + ない.

Wrong / corrected pairs

✗ 行きないです。
✓ 行きません。 / 行かないです。
Polite negative is ません; plain negative of 行く is 行かない (NOT 行きない).
✗ たべるない。
✓ たべない。
Group-2: drop る then add ない. 食べる→食べない.
✗ こない (intent: "did not come" past)
✓ こなかった。
こない is present-negative ("does not come"); past-negative is こなかった.
