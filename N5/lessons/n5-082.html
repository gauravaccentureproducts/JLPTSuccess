い-Adjective past negative: 〜くなかったです / 〜くありませんでした

ENい-Adjective past negative

JAい-けいようしの かこ ひてい

Explanation

Drop い, add くなかった. Polite form adds です. Alternative: くありませんでした (more formal, less common in speech).

Attaches to

i_adjective

Examples

JAきのうは さむくなかったです。
ENYesterday wasn't cold.
JAその えいがは おもしろくなかったです。
ENThat movie wasn't interesting.
JAテストは むずかしくなかったですか。
ENWasn't the test difficult?
JAりょうりは おいしくありませんでした。
ENThe meal wasn't tasty.
JAテストは むずかしくなかったです。
ENThe test wasn't difficult.
JAホテルは あまり よくなかったです。
ENThe hotel was not very good.
JAあの えいがは よくなかったですか。
ENWasn't that movie any good?
JAきのうは あまり あつくなかったです。
ENIt wasn't very hot yesterday.
JAあの えいがは おもしろくなかったです。
ENThat movie wasn't interesting.
JAりょうりは おいしくなかったです。
ENThe food wasn't tasty.

Common mistakes

✗ あついじゃありませんでした
✓ あつくなかったです
い-adj never use じゃありません or でした. Build past-negative from the adjective itself.
✗ いいくなかった
✓ よくなかった
いい (good) is irregular: past-negative is よくなかった.
✗ きのうは さむいくなかったです。
✓ きのうは さむくなかったです。
い-adjective past negative: drop い, add くなかった, then です. さむい → さむ + くなかった + です. Don't double-mark with い+くなかった. Pattern: stem + くなかった + です.

Wrong / corrected pairs

✗ 高いじゃ ありませんでした。
✓ 高くなかったです。 / 高くありませんでした。
い-adjective past-negative uses くなかった (NOT じゃ-form).
✗ たかいくなかったです。
✓ たかくなかったです。
Drop い + くなかった: 高い→高くなかった.
✗ たかくありませんかった。
✓ たかくありませんでした。
Polite past-negative is くありませんでした (not くありませんかった); the past tense lives in でした.
