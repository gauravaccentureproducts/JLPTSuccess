この／その／あの／どの ＋ N (noun)

ENAdjective form: this / that / which + Noun (always followed by a noun)

JA「この・その・あの・どの」+ めいし で、ものを 「これ」と さして いいます。「この本」「その本」「あの本」「どの本」。

Explanation

Use この/その/あの/どの directly before a noun. Mirrors これ/それ/あれ/どれ exactly but ALWAYS followed by a noun: この ほん = 'this book', あの 人 = 'that person'.

Attaches to

before_noun

Examples

JAこの ほんを よみました。
ENI read this book.
JAその ペンは あなたのですか。
ENIs that pen yours?
JAあの 人は だれですか。
ENWho is that person?
JAどの くるまが いいですか。
ENWhich car is good?
JAこの みせは あまり たかくないです。
ENThis shop isn't very expensive.
JAあの ねこは どこから 来ましたか。
ENWhere did that cat come from?
JAどの えいがを 見ますか。
ENWhich movie will you watch?
JAこの ほんは おもしろいです。
ENThis book is interesting.
JAその ペンは だれの ですか。
ENWhose pen is that?
JAあの くるまは はやいです。
ENThat car is fast.

Common mistakes

✗ これの ほんを よみます。
✓ この ほんを よみます。
Don't write これの ほん. Use この directly before a noun. これ + noun is wrong - それ is for 'that pronoun', この is for 'this + noun'.
✗ この。
✓ この + (a noun) - always.
この/その/あの/どの can NEVER stand alone - they MUST be followed by a noun. If standing alone, use これ/それ/あれ/どれ.
✗ これ ほんを よみました。
✓ この ほんを よみました。
これ is a STANDALONE pronoun ('this thing'). この is a NOUN-MODIFIER ('this [noun]'). Before a noun, use この/その/あの/どの. Without a noun, use これ/それ/あれ/どれ. Common N5 error: skipping the の of この.

Wrong / corrected pairs

✗ この は ほんです。 (intent: this is a book)
✓ これは ほんです。
この must be followed by a noun (この + N). For "this (alone)" use これ.
✗ この ほんが わたし の です。
✓ この ほんは わたしのです。
When introducing a topic ("As for this book, it is mine"), use は, not が.
✗ そのほん。 (no spacing)
✓ その ほん。
Wakachi spacing: separate determiner from noun in early-N5 reading.
