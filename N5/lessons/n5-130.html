〜に〜をあげます

ENあげる / あげます - give (to others)

JA「Aは Bに Xを あげます」で、Aから Bに ものを わたします。「私は 友だちに 本を あげました」。

Explanation

あげる = give from speaker (or in-group) to others. Recipient marked with に. Don't use あげる when receiving from someone - use もらう.

Attaches to

verb

ENI gave a book to my friend.

Common mistakes

✗ わたしは 田中さんが 本を あげました。
✓ わたしは 田中さんに 本を あげました。
Recipient is marked with に, not が. Pattern: [giver] は [recipient] に [object] を あげる.
✗ ともだちを 本を あげました。
✓ ともだちに 本を あげました。
あげる/さしあげる take RECIPIENT に + OBJECT を. Double-を is wrong. Pattern: [giver]は + [recipient]に + [object]を + あげる. The recipient particle is に, not を.
✗ ともだちを 本を あげました。
✓ ともだちに 本を あげました。
Giving verb あげる: recipient takes に, object takes を. Pattern: [giver]は + [recipient]に + [object]を + あげる. Don't double を; に for recipient is mandatory.
✗ ともだちに 本を やりました。 (to a peer/friend)
✓ ともだちに 本を あげました。
やる ('give') is for inferiors/pets/plants — using it with a peer-level friend sounds rude or childish. For peer giving, use あげる. For superiors, use さしあげる (humble).

Wrong / corrected pairs

✗ 友だちを 本を あげました。
✓ 友だちに 本を あげました。
Recipient takes に; only the gift takes を.
✗ ともだちに ほんを くれました。 (intent: I gave my friend a book)
✓ ともだちに ほんを あげました。
あげる = "I/X give to Y"; くれる = "Y give TO me/in-group". Direction matters.
✗ はは が プレゼントを あげました。 (intent: my mother gave me a gift)
✓ はは が プレゼントを くれました。
Mother giving TO me (in-group recipient) = くれる, not あげる. あげる would mean mother gave to someone else.
