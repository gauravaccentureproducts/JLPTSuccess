どうして〜か。〜から。

ENどうして〜か。〜から。 - Q-and-A reason pattern

JA「Verb-て ください」は ていねいに たのむ ことばです。「まって ください」「見て ください」。

Explanation

Standard pattern: question with どうして, answer using から. The answer often ends 〜からです (because + polite).

Attaches to

dialogue

Examples

JAどうして 来ませんでしたか。―あたまが いたかったからです。
ENWhy didn't you come? - Because I had a headache.
JAどうして 行きませんでしたか。―ねつが あったからです。
ENWhy didn't you go? - Because I had a fever.
JAどうして 学校を 休みましたか。―あたまが いたかったからです。
ENWhy did you skip school? - Because I had a headache.
JAどうして 学校を 休みましたか。― あたまが いたかったからです。
ENWhy did you skip school? — Because I had a headache.
JAどうして 日本語を べんきょうしますか。 すきだからです。
ENWhy do you study Japanese? Because I like it.
JAどうして うちに かえりますか。 つかれたからです。
ENWhy are you going home? Because I'm tired.
JAどうして かいませんでしたか。 たかかったからです。
ENWhy didn't you buy it? Because it was expensive.
JAどうして きませんでしたか。 あめでしたから。
ENWhy didn't you come? Because it was raining.
JAどうして すきですか。 おいしいからです。
ENWhy do you like it? Because it's tasty.
JAどうして いそぎますか。 おそいからです。
ENWhy are you hurrying? Because it's getting late.

Common mistakes

✗ どうして 来ませんでしたか。―ねつが あって。
✓ どうして 来ませんでしたか。―ねつが あったからです。
Reason answers in formal speech end in 〜からです, not just the bare clause.
✗ どうして 来ませんでしたか。―あたまが いたかったから。
✓ どうして 来ませんでしたか。―あたまが いたかったからです。
Answering どうして with から requires です at the end in polite register: 'because [reason] です'. Pattern: あたまが いたかったから + です. Without です, the answer is too casual for the polite question.
✗ どうして 来ませんでしたか。―あたまが いたいでした からです。
✓ どうして 来ませんでしたか。―あたまが いたかったからです。
i-adj past polite: drop い, add かった + です → いたかったです. Then attach から (because) + です for explanation form. Don't insert でした before からです — that's wrong for i-adj.

Wrong / corrected pairs

✗ どうして いきますか。 おそい から です。
✓ どうして いかないんですか。 おそいからです。
For "why" + reason explanation often use 〜んです (explanatory). Plain Vます is too neutral.
✗ どうして きません か。 おそかった です から。
✓ どうして きませんでしたか。 おそかったからです。
Tense agreement: past Q needs past A; おそかった + から (reason past) → からです at end.
✗ なぜから、いきます。
✓ なぜなら、いきます。
Reason connector is なぜなら (formal/written), not なぜから.
