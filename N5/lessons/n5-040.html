この / その / あの / どの + Noun

ENAdjectival demonstratives + noun

JA「この・その・あの・どの」+ めいし で、ものを さします。「この本」「その本」「あの本」「どの本」。

Explanation

Use この/その/あの/どの directly before a noun. Mirrors これ/それ/あれ/どれ exactly but ALWAYS followed by a noun: この ほん = 'this book', あの 人 = 'that person'.

Examples

JAこの ほんを よみました。
ENI read this book.
JAその ペンは あなたのですか。
ENIs that pen yours?
JAあの 人は だれですか。
ENWho is that person?
JAどの くるまが いいですか。
ENWhich car is good?
JAこの みせは あまり たかくないです。
ENThis shop isn't very expensive.
JAこの りょうりは おいしいです。
ENThis dish is delicious.
JAあの 大きい たてものは 学校です。
ENThat big building is the school.
JAこの まちは ちいさいです。
ENThis town is small.
JAその くるまは あたらしいです。
ENThat car (near you) is new.
JAあの ひとは だれですか。
ENWho is that person?

Common mistakes

✗ これ 本は おもしろいです。
✓ この 本は おもしろいです。
これ is a standalone pronoun; cannot precede a noun. Use この + Noun for "this X".
✗ それの 本
✓ その 本
No の after その/この/あの/どの when a noun directly follows.
✗ これ ほんを よみました。
✓ この ほんを よみました。
Same noun-modifier rule as n5-015. この/その/あの require a following noun. これ/それ/あれ stand alone. Mixing forms is the typical mistake.

Wrong / corrected pairs

✗ このは ほんです。
✓ これは ほんです。 / この ほんは ...
この MUST take a noun directly. For pronoun use これ.
✗ その なほん。
✓ その ほん。
Determiner + noun directly. Do not insert な (this is for na-Adj only).
✗ どの ほんは いいですか。
✓ どの ほんが いいですか。
Question with どの takes が on the resulting NP.
