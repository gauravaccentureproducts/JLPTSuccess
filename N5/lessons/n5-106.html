Noun + が ほしいです

ENNoun + が ほしいです (same as n5-101 but with explicit noun emphasis)

JA「Noun + が ほしいです」で、ほしい ものを いいます。「車が ほしいです」「あたらしい かばんが ほしいです」。を ではなく が を つかう ことが ポイントです。

Explanation

Use ほしい to want a NOUN. Same rule as n5-101: subject (the wanted thing) takes が.

Attaches to

noun

Examples

JA新しい かばんが ほしいです。
ENI want a new bag.
JAおかねが ほしいです。
ENI want money.
JA休みが ほしいです。
ENI want a day off.
JAたんじょうびに とけいが ほしいです。
ENFor my birthday I want a watch.
JA新しい くるまが ほしいです。
ENI want a new car.
JAあたらしい かばんが ほしいです。
ENI want a new bag.
JAおおきい いえが ほしいです。
ENI want a big house.
JAあたらしい パソコンが ほしいです。
ENI want a new computer.
JAなにが いちばん ほしいですか。
ENWhat do you want most?
JAじてんしゃが ほしいです。
ENI want a bicycle.

Common mistakes

✗ わたしは 新しい かばんを ほしいです。
✓ わたしは 新しい かばんが ほしいです。
ほしい takes が, not を. The desired thing is the が-marked target.
✗ 田中さんは 車が ほしいです。
✓ 田中さんは 車を ほしがって います。
ほしい can only describe SPEAKER's desire. For third-person desires, use ほしがる.
✗ 新しい かばんを ほしいです。
✓ 新しい かばんが ほしいです。
ほしい takes が, not を. Same rule as preference adjectives — the wanted thing is the subject of desire, not a transitive object. Always: NOUN + が + ほしい/ほしくない/ほしかった/etc.
✗ 新しい かばんを ほしいです。
✓ 新しい かばんが ほしいです。
ほしい takes が for the wanted-thing (not を). Same family rule as すき, じょうず. The desired-noun is subject of want, not transitive object.

Wrong / corrected pairs

✗ くるまを ほしいです。
✓ くるまが ほしいです。
ほしい takes が. (Like other feeling adjectives.)
✗ ともだちは くるまが ほしいです。
✓ ともだちは くるまを ほしがっています。
ほしい works for 1st person; 3rd-person needs ほしがる (observable wanting).
✗ ほしいくないです。
✓ ほしくないです。
い-Adj negative drops い: ほしい→ほしく+ない.
