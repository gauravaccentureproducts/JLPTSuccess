〜にします

EN〜にします - decide on / choose

JA「〜にします」で、えらぶ ことや きめる ことを いいます。「コーヒーに します」「あおい シャツに します」。

Explanation

〜にする = 'decide on / choose / make it ~'. Common in restaurants for ordering: コーヒーにします = 'I'll have coffee'. The に marks the choice. Contrast with 〜になる (n5-143), which is non-volitional change.

Attaches to

noun

Examples

JAコーヒーに します。
ENI'll have coffee.
JAなにに しますか。
ENWhat will you have?
JAのみものは おちゃに します。
ENI'll have tea for the drink.
JAばんごはんは カレーに します。
ENFor dinner I'll have curry.
JAジュースに します。
ENI'll have juice.
JAばんごはんは てんぷらに します。
ENI'll have tempura for dinner.
JAコーヒーに しますか、おちゃに しますか。
ENWill you have coffee or tea?
JAおちゃに します。
ENI'll have tea.
JAのみものは なにに しますか。
ENWhat will you have to drink?
JAこれに します。
ENI'll choose this one.

Common mistakes

✗ コーヒーを します。
✓ コーヒーに します。
"Choose / decide on X" uses に, not を. を します means "do X" (action verb).
✗ コーヒーを します。
✓ コーヒーに します。
〜にします = 'I'll have/choose [noun]' (deliberate selection). The particle is に, not を. Without に, the sentence is structurally incomplete in this set-pattern frame. Standard ordering choice in restaurants: NOUN + に + します.
✗ コーヒーを します。
✓ コーヒーに します。
〜にします = 'decide on / choose [noun]'. The pattern requires に before します. を would treat コーヒー as a transitive object of する ('do coffee'), which is nonsense. For 'I'll have coffee', use NOUN + に + します.

Wrong / corrected pairs

✗ コーヒーを します。
✓ コーヒーに します。
Choice/decision pattern is N + に + する. を marks general object.
✗ にします か コーヒーを。
✓ コーヒーに しますか。
SOV order: object first, then にします.
✗ コーヒーで します。 (intent: I will choose coffee)
✓ コーヒーに します。
Decision particle is に, not で.
