な-Adjective past negative: 〜じゃありませんでした

ENな-Adjective past negative

JAな-けいようしの かこ ひてい

Explanation

Past-negative for な-adj: + じゃありませんでした (or formal ではありませんでした). Same pattern as nouns.

Attaches to

na_adjective

Common mistakes

✗ しずかくなかった
✓ しずかじゃありませんでした
な-adjective past negative: stem + じゃありませんでした (or じゃなかった for plain). The form くなかった is for い-adjective only (おいしくなかった). Don't apply i-adj rules to na-adj.
✗ きのうの しけんは かんたんじゃありませんかった。
✓ きのうの しけんは かんたんじゃありませんでした。
な-adjective past negative: stem + じゃありません + でした (or じゃなかった for plain). The form じゃありませんかった is wrong — でした follows ません, not かった.
✗ きのうの しけんは かんたんくなかったです。
✓ きのうの しけんは かんたんじゃありませんでした。
な-adj past negative: stem + じゃありませんでした. Don't apply い-adj past-negative くなかった to な-adj. Pattern: な-adj-stem + じゃありませんでした.

Wrong / corrected pairs

✗ しずかかったじゃありません。
✓ しずかじゃありませんでした。
na-Adj past-negative is じゃありませんでした (not かった+じゃありません).
✗ すきくなかったです。
✓ すきじゃありませんでした。 / すきじゃなかったです。
すき is na-Adj. Use じゃ(あり)ませんでした, not くなかった.
✗ げんきじゃありませんでした と、おもいます。
✓ げんきじゃありませんでしたと、おもいます。
No space before quotative と.
