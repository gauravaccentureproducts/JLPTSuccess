Verb-てもいいです

ENVerb-てもいいです - may / it's OK to do (permission)

JA「〜てもいいです」(きょか)

Explanation

Attach も + いいです to the て-form to give or ask permission: 'may I / it's okay to do X'. Often used as a question (てもいいですか).

Attaches to

verb_te

Examples

ENMay I take a photo here?
JAもう かえってもいいです。

JAえんぴつで かいても いいですよ。
ENIt's OK to write in pencil.
JAすこし やすんでも いいですか。

Common mistakes

✗ たべるもいいです
✓ たべてもいいです
Permission 'may do X' uses V-て + も + いい: たべる → たべて → たべてもいい. Don't keep the dictionary form る — the て-form is the required derivation. Pattern: V-te + も + いい (+ ですか for question).
✗ ここで しゃしんを とるても いいですか。
✓ ここで しゃしんを とってもいいですか。
Group-1 verb とる ('take'): te-form is とって (る → って sub-rule). Then + も + いい = permission ('is it OK to take photos?'). Don't preserve る in the te-form chain. Full: V-te + も + いい.
✗ ここで しゃしんを とるても いいですか。
✓ ここで しゃしんを とってもいいですか。
とる (Group 1, ends in る but う-row pattern): te-form = って (drop る, add って). The form とるて is wrong. Pattern: V-te + も + いい + ですか for permission-question.
✗ ここで しゃしんを とってもいいか？(formal context)
✓ ここで しゃしんを とってもいいですか。
Drop the bare か for casual; add ですか for polite. In formal contexts (museums, restaurants, business), use the full polite form ですか.

Wrong / corrected pairs

✗ たべてもいいか。 (to teacher)
✓ たべてもいいですか。
Permission-asking to a senior must keep です; dropping です is too casual.
✗ たべるても いいです。
✓ たべても いいです。
Drop る to form て first: 食べる→食べて+も+いいです.
✗ たべて いいです。
✓ たべても いいです。
The も is required for permission "even doing X is OK". Without も, the sentence is ungrammatical for permission.
