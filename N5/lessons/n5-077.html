Verb-ないでください

ENVerb-ないでください - please don't do

JA「Verb-ないで ください」は 「しないで」と たのむ ことばです。「ここで たばこを すわないで ください」。

Explanation

ないでください = polite request not to do. Take the plain ない-form, add でください. Politeness pair with てください (polite affirmative request). Casual counterparts: drop ください for ないで (negative) and 〜て (affirmative).

Attaches to

verb_nai

Examples

JAここで しゃしんを とらないでください。
ENPlease don't take photos here.
JAわすれないでください。
ENPlease don't forget.
JAおおきい こえで 話さないでください。
ENPlease don't speak loudly.
JAあぶないですから、はいらないでください。
ENIt's dangerous, so please don't enter.
JAここで たばこを すわないでください。
ENPlease don't smoke here.
JAへやで さわがないでください。
ENPlease don't make noise in the room.
JAおそく 来ないでください。
ENPlease don't come late.
JAここで しゃしんを とらないで ください。
ENPlease don't take pictures here.
JAおおきい こえで はなさないで ください。
ENPlease don't speak loudly.
JAまだ いかないで ください。
ENPlease don't go yet.

Common mistakes

✗ たべませんでください
✓ たべないでください
Use the plain ない-form, not the polite ません-form.
✗ ここで しゃしんを とらないで ください。
✓ ここで しゃしんを とらないでください。
Negative request 'please don't': V-ない + で + ください. The form とらないでください is correct as one phrase. Don't separate ないで from ください — they form a tight grammatical unit. Pattern: V-ない (drop い) + で + ください.
✗ ここで しゃしんを とらないて ください。
✓ ここで しゃしんを とらないでください。
Negative request: V-ない form + で + ください. The form ないて (te-form of ない) is wrong — ない attaches to で (not て). Pattern: V-stem (あ-row) + ない + で + ください.
✗ ここで しゃしん とらないでください。
✓ ここで しゃしんを とらないでください。
Object marker を required even in the negative-request frame. しゃしんを とる is the verb phrase; とる takes を. Don't drop in formal speech.

Wrong / corrected pairs

✗ たべるないで ください。
✓ たべないで ください。
Group-2: drop る then ない: 食べる→食べない→食べないで.
✗ たべなくて ください。
✓ たべないで ください。
Negative request uses ないで, not なくて. なくて is for reasons/causes.
✗ いかないでください。 (to a friend casually)
✓ いかないで。
Plain casual register drops ください: ないで alone is fine with intimates.
