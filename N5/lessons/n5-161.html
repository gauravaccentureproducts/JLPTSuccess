Noun + の + まえに

ENNoun の まえに - see 〜まえ

JA「Nounの まえに」は 「その ことの まえ」と いう いみです。「ごはんの まえに 手を あらいます」。

Explanation

まえ = 'before'. Noun + の + まえ(に); Verb-plain (NON-PAST) + まえに. Note: verb stays NON-PAST even when the meaning is 'before doing X (in the past)'.

Examples

JAごはんの まえに てを あらいます。
ENI wash my hands before meals.
JAねる まえに 本を 読みます。
ENI read a book before going to bed.
JAばんごはんの まえに 休みました。
ENI rested before dinner.
JAテストの まえに たくさん べんきょうしました。
ENI studied a lot before the test.
JA三日 まえに 来ました。
ENI came three days ago.
JAドアの まえに ねこが います。
ENA cat is in front of the door.
JAねる まえに しゅくだいを します。
ENI'll do homework before bed.
JAしごとの まえに、 コーヒーを のみます。
ENBefore work, I drink coffee.
JAごはんの まえに、 てを あらいます。
ENBefore meals, I wash my hands.
JAにじの まえに きて ください。
ENPlease come before 2.

Common mistakes

✗ ごはん まえに てを あらいます。
✓ ごはんの まえに てを あらいます。
Noun + まえ requires の. (Verb-plain + まえに does NOT take の: 食べる まえに.)
✗ ごはんの まえで てを あらいます。
✓ ごはんの まえに てを あらいます。
Noun + の + まえに = 'before [noun]' (using に). The form まえで doesn't fit — まえ takes に as its companion particle (parallel to あと + で which is the after-companion).
✗ ごはんに まえに てを あらいます。
✓ ごはんの まえに てを あらいます。
Noun + の + まえに. The connector before まえに is の. に would mean 'to the meal', changing meaning. For 'before meals', use ごはん + の + まえに.

Wrong / corrected pairs

✗ がっこうまえに、 たべます。
✓ がっこうのまえに、 たべます。
Noun + まえに needs の.
✗ がっこうのまえで、 たべます。 (intent: before going to school)
✓ がっこうのまえに、 たべます。
まえに for "before (time)"; まえで is a different meaning ("in front of").
✗ にじのまえで、 きました。 (time intent)
✓ にじのまえに、 きました。
Time-before takes に, not で.
