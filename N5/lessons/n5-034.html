しか〜ない

ENしか〜ない - only (with negative verb)

JA「〜しか〜ない」(げんてい)

Explanation

しか REQUIRES a negative verb to mean 'only'. The polarity is grammatically negative but the meaning is affirmative-with-emphasis ('only X, nothing more').

Attaches to

noun

Examples

JA1,000円しか ありません。
ENI only have 1000 yen.
JAわたししか いません。
ENOnly I am here.
JAりんごは 一つしか ありません。
ENThere is only one apple.
JAパーティーには 三人しか 来ませんでした。
ENOnly three people came to the party.
JAおかねは 100円しか ありません。
ENI have only 100 yen.
JAペンは 一本しか ありません。
ENI have only one pen.
JAおちゃしか のみません。
ENI drink only tea.
JAいちじかんしか ねませんでした。
ENI slept only one hour.
JAひとりしか きませんでした。
ENOnly one person came.
JAひゃくえんしか ありません。
ENI have only 100 yen.

Common mistakes

✗ 1,000円しか あります。
✓ 1,000円しか ありません。
しか requires the verb to be negative.
✗ 1,000円しか あります。
✓ 1,000円しか ありません。
しか REQUIRES a negative predicate (ありません, できません, わかりません). It means 'nothing but / only X' with a regretful nuance. しか + positive predicate (あります) is grammatically wrong — always pair with ない/ません.
✗ 1,000円しか あります。
✓ 1,000円しか ありません。
しか mandates a negative predicate. 'しか + ある' is structurally impossible — must be 'しか + ない'. The pattern conveys 'nothing but X' with a regretful nuance; positive predicates can't carry that nuance.

Wrong / corrected pairs

✗ いちじかんしか べんきょうしました。
✓ いちじかんしか べんきょうしませんでした。
しか REQUIRES a negative predicate. Without ません/ない the sentence is ungrammatical.
✗ ひゃくえんしか もって いません。 (intent: I have ¥100)
✓ ひゃくえんしか ありません。
For money/possessions in this idiom use あります not もっている; both work but あります is standard for しか.
✗ にんげんが ひとりだけしか いませんでした。
✓ ひとりしか いませんでした。
Do not stack だけ + しか. Pick one (しか + negative or だけ + positive).
