Verb-ない + ほうがいい

ENVerb-ない + ほうが いい - shouldn't do

JA「〜ないほうが いい」

Explanation

Negative version of n5-170: 'you'd better not'. Full form: Verb-ない + ほうがいい. Example: たばこを すわないほうがいい = 'you'd better not smoke'. Same advisory register as the affirmative; same softening with です.

Attaches to

verb_nai

Common mistakes

✗ 行かなく ほうが いいです。
✓ 行かない ほうが いいです。
Negative ほうがいい uses the plain ない form (行かない), not the て-form ない (行かなくて).
✗ あまり 食べたほうが いいです。
✓ あまり 食べないほうが いいです。
Negative advice uses V-ない + ほうがいい. The form 食べたほうがいい (positive) means 'better to eat'. For 'better NOT to eat much', use 食べない + ほうがいい. Pattern: V-ない + ほうがいい.
✗ あまり 食べたほうが いいです。
✓ あまり 食べないほうが いいです。
Negative advice: V-ない + ほうがいい. 食べたほうがいい (positive) means 'better to eat'. For 'better NOT to eat much', use 食べない + ほうがいい. Don't confuse positive and negative advice.

Wrong / corrected pairs

✗ たべない ほうが いいですね。 (commanding tone)
✓ たべない ほうが いいですよ。
Advice typically uses よ (informing); ね (agreement-seeking) sounds odd here.
✗ たべないて ほうがいい。
✓ たべない ほうがいい。
Vない directly + ほうがいい (no て-form).
✗ たべない ほうが いい です だ。
✓ たべない ほうが いいです。
いい + です terminates; do not add だ.
