〜がほしいです

EN〜が ほしいです - want (an object)

JA「〜が ほしいです」で ほしい ものを いいます。「車が ほしいです」。

Explanation

ほしい is an い-adjective meaning 'want'. The desired thing takes が. Limited to first person (or in questions, second person).

Attaches to

noun

Examples

JA新しい くるまが ほしいです。
ENI want a new car.
JAなにが ほしいですか。
ENWhat do you want?
JAなにも ほしくないです。
ENI don't want anything.
JAあたらしい かばんが ほしいです。
ENI want a new bag.
JAあたらしい くつが ほしいです。
ENI want new shoes.
JAおちゃが ほしいです。
ENI want some tea.
JAもっと 時間が ほしいです。
ENI want more time.
JAあたらしい くるまが ほしいです。
ENI want a new car.
JAなにが いちばん ほしいですか。
ENWhat do you want most?
JAいまは なにも ほしくないです。
ENI don't want anything right now.

Common mistakes

✗ 田中さんは あたらしい くるまが ほしいです。
✓ 田中さんは あたらしい くるまを ほしがって います。
Third-person desires use ほしがっている (showing-signs-of-wanting), not direct ほしい. Direct ほしい is for the speaker's own / second-person-asked desire only.
✗ 新しい くるまを ほしいです。
✓ 新しい くるまが ほしいです。
ほしい ('want, desire') takes が, like すき/じょうず. The desired thing is the subject of want, not a direct object. Common N5 error: applying transitive を to ほしい. Always: NOUN + が + ほしい.
✗ 新しい くるまを ほしいです。
✓ 新しい くるまが ほしいです。
ほしい ('want, desire') uses が for the wanted-thing. Same rule as すき, じょうず, わかる. The desired-noun is the subject of the wanting-state, not a transitive object.

Wrong / corrected pairs

✗ 本を ほしいです。
✓ 本が ほしいです。
ほしい takes が because the wanted-thing is the SUBJECT.
✗ わたしは くるまを ほしいです。
✓ わたしは くるまが ほしいです。
ほしい takes が. を is wrong for desired-object marking with adjectives of feeling.
✗ せんせいは くるまが ほしいです。
✓ せんせいは くるまを ほしがっています。
ほしい with 1st-person OK; 3rd-person requires ほしがっている (observable wanting-behavior).
