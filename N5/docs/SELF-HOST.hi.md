# स्व-होस्टिंग JLPTSuccess

यह गाइड JLPTSuccess N5 सब-ऐप को अपने इंफ्रास्ट्रक्चर से चलाने के लिए है — व्यावसायिक स्कूल, भाषा संस्थान, NGO, या किसी भी संस्था के लिए जो ऑफ़लाइन-सक्षम, खाता-रहित JLPT N5 तैयारी टूल अपने ब्रांड में चाहती है।

> यह `docs/SELF-HOST.md` (अंग्रेज़ी) का हिंदी संस्करण है — संक्षिप्त रूप में। पूरी तकनीकी गाइड के लिए मूल अंग्रेज़ी फ़ाइल देखें।

## संक्षिप्त निर्देश

```bash
git clone https://github.com/gauravaccentureproducts/JLPTSuccess.git
cd JLPTSuccess
python -m http.server 8000
# ब्राउज़र में: http://localhost:8000/N5/
```

सब कुछ ब्राउज़र में चलता है — कोई बैकेंड नहीं, कोई डेटाबेस नहीं, कोई API कुंजी नहीं। ऐप ~1 MB शेल + ज़रूरत-पर-लोड ऑडियो है। उत्पादन तैनाती GitHub Pages, Netlify, Vercel, Cloudflare Pages, या किसी nginx/Apache पर।

## ब्रांड अनुकूलन

तीन परतें — हर अगली अधिक आक्रामक:

### परत 1 — रनटाइम थीम ओवरराइड (पसंदीदा)

`data/theme-overrides.json` फ़ाइल को CSS कस्टम-प्रॉपर्टी मैपिंग्स के साथ छोड़ दें। ऐप शुरू होने पर लोड होता है; `css/main.css :root` में परिभाषित डिज़ाइन टोकन को ओवरराइड करता है।

उदाहरण `data/theme-overrides.json`:

```json
{
  "tokens": {
    "--color-accent": "#1e6091",
    "--color-bg":     "#ffffff",
    "--color-text":   "#212529"
  },
  "brand": {
    "name":      "एक्मे लैंग्वेज स्कूल",
    "short_name": "एक्मे N5",
    "logo_url":  "../assets/logo/acme-mark.svg"
  }
}
```

रंग / फ़ॉन्ट / ब्रांड-नाम स्वैप के लिए कोई कोड बदलाव नहीं चाहिए।

### परत 2 — फ़ोर्क-विशिष्ट लोगो + manifest स्वैप

अलग favicon / install आइकन के लिए:

1. `assets/logo/icon-192.png`, `icon-512.png`, `mark.svg` को अपने फ़ोर्क में बदलें।
2. यदि परत 1 पर्याप्त नहीं तो `N5/manifest.webmanifest` में `name` + `short_name` अपडेट करें।

### परत 3 — पूर्ण स्रोत फ़ोर्क

बड़े बदलावों (नए सर्फ़ेस, अलग सामग्री-कोष आदि) के लिए रिपॉजिटरी फ़ोर्क करें और सीधे संपादित करें। MIT लाइसेंस इसकी अनुमति देता है।

## तैनाती लक्ष्य

- **GitHub Pages**: फ़ोर्क करें → सेटिंग्स → पेज → स्रोत: master शाखा से तैनात करें / `/` (रूट)। पहली तैनाती ~30 सेकंड।
- **Netlify / Vercel / Cloudflare Pages**: रिपॉजिटरी डैशबोर्ड में डालें। बिल्ड कमांड कोई नहीं। प्रकाशन डायरेक्टरी: `/`।
- **nginx / Apache** (संस्थागत इंट्रानेट): स्थैतिक फ़ाइलें परोसें। हैश-राउटिंग के कारण SPA-redirect सेटअप ज़रूरी नहीं।

## सामग्री अनुकूलन

`N5/data/{grammar,vocab,kanji,reading,listening}.json` 5 कोष फ़ाइलें संपादन-योग्य सामग्री-सतह हैं। उनकी स्कीमा अक्षुण्ण रखें (रनटाइम विशिष्ट फ़ील्ड-नाम अपेक्षित करता है — देखें `N5/tools/check_content_integrity.py` 48 invariants के लिए), पर वास्तविक प्रविष्टियाँ CC BY-SA हैं — अपने पाठ्यक्रम के अनुसार अनुवाद, अनुकूलन, या विस्तार करें।

बदलाव के बाद चलाएँ:

```bash
python N5/tools/check_content_integrity.py
python N5/tools/build_version_json.py    # data/version.json + sw.js bump
```

48 इन्वैरिअंट्स की जाँच रिलीज़ गेट है; यदि विफल हो, तो तैनाती नहीं भेजनी चाहिए।

## अनुवाद (बहुभाषी तैनाती)

2 लोकेल फ़ाइलें `N5/locales/{en,hi}.json` पर (2026-05-06 को 5 से 2 तक संकुचित — देखें CHANGELOG v1.12.40)। रनटाइम पहली यात्रा पर `navigator.language` (और `navigator.languages[]`) का पता लगाकर निकटतम मिलान चुनता है। अपने फ़ोर्क के लिए नया लोकेल जोड़ने के लिए:

1. `en.json` को `<lc>.json` में कॉपी करें जहाँ `<lc>` BCP-47 बेस है (जैसे `pt`, `ar`)।
2. कुंजियों का अनुवाद करें।
3. `N5/js/i18n.js` के `SUPPORTED` ऐरे में `<lc>` जोड़ें।
4. वैकल्पिक रूप से सामग्री बॉडी (व्याकरण व्याख्या, शब्दावली अर्थ, कान्जी अर्थ) का अनुवाद करें — देखें `N5/docs/TRANSLATING.md` प्रति-लोकेल सामग्री-फ़ील्ड कार्यप्रवाह के लिए।

## निजता-रुख

ऐप सामान्य संचालन में शून्य नेटवर्क कॉल करता है। `N5/index.html:18` पर CSP केवल समान-मूल लागू करता है। localStorage `jlpt-n5-tutor:` के तहत नामस्थानित है। कोई कुकीज़ नहीं, कोई टेलीमेट्री नहीं, कोई तृतीय-पक्ष स्क्रिप्ट नहीं।

यदि आप फ़ोर्क करके एनालिटिक्स जोड़ते हैं, तो **`N5/PRIVACY.md` अपडेट करें** ताकि बदलाव दिखे। निजता का दावा एक niche-defining सुविधा है; मौन रूप से टेलीमेट्री जोड़ना उस भरोसे को नष्ट करता है जिसके आसपास ऐप स्थापित है।
