〜ないといけない

ENMust do (common spoken variant)

JA「〜ないと いけない」は ぎむを いいます。「〜なくては いけない」の カジュアルな かたちです。「もう ねないと いけない」「べんきょうを しないと いけない」。

Explanation

Negative て-form + は + いけない = 'must do' (lit. 'not doing won't work'). Casual contraction: 〜なくちゃ.

Examples

JAはやく かえらないと いけない。
ENI must go home soon.
JAもう 行かないと いけません。
ENI have to go now.
JAしゅくだいを しないと いけない。
ENI have to do my homework.
JAはやく 行かないと いけません。
ENI have to go quickly.
JAはやく ねないと いけません。
ENI have to sleep early.
JAべんきょうしないと いけません。
ENI have to study.
JAお金を はらわないと いけません。
ENI have to pay the money.
JAもう いかないと いけません。
ENI have to go now.
JAやさいを たべないと いけません。
ENI have to eat vegetables.
JAおかねを はらわないと いけません。
ENI have to pay.

Common mistakes

✗ しゅくだいを しないと なりません。
✓ しゅくだいを しないと いけません。
〜ないと idiomatically pairs with いけない/いけません (more colloquial), not ならない. (See n5-174.)
✗ はやく かえらないと いけませんない。
✓ はやく かえらないと いけない。
Obligation: V-ない + と + いけない (NOT と + いけませんない). The two are separate units. Pattern: V-ない + と + いけない/だめ. Polite: いけません. Don't double-negate with ないと いけませんない.
✗ はやく かえらなくては いけない。 (using なくては where ないと expected)
✓ はやく かえらないと いけない。
n5-175 specifically: V-ない + と + いけない. The form 〜なくては (which uses くて) is the n5-173/174 pattern. Different sub-patterns — use the と-form for this specific frame.

Wrong / corrected pairs

✗ たべないと いける。
✓ たべないと いけない。
Negative いけない is the obligation marker; positive いける changes meaning.
✗ たべないとも いけない。
✓ たべないと いけない。
Particle と alone; do not add も.
✗ たべると いけない。
✓ たべないと いけない。
Obligation pattern uses NEGATIVE Vない + と + いけない. Plain Vる changes to "if I eat, it is no good".
