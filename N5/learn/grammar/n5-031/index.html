〜の

ENSentence-final の as informal question marker (plain-form + の?)

JAふつうけい + の?

Explanation

After plain-form verbs and adjectives, sentence-final の forms a casual / familiar question. Used with friends and family. Don't use this in formal contexts; use か with です/ます-form instead. Pairs with n5-012 (formal か).

Examples

JAどこへ いくの？
ENWhere are you going?
JAなにを たべるの？
ENWhat are you eating?
JAごはん 食べないの？
ENAren't you going to eat?
JAきのう どこに いたの？
ENWhere were you yesterday?
JAどこへ 行くの？
ENWhere are you going?
JAなに、食べているの？
ENWhat are you eating?
JAなんで そう おもうの？
ENWhy do you think so?
JAあかいのを ください。
ENPlease give me the red one.
JAおおきいのが すきです。
ENI like big ones.
JAもっと やすいのは ありますか。
ENDo you have a cheaper one?

Common mistakes

✗ あした きますの？
✓ あした くるの？
Informal の attaches to PLAIN form, not polite ます-form.
✗ どこへ いくですか？
✓ どこへ いくの？
Sentence-final の (casual) substitutes for ですか in casual register. The form いくですか doesn't exist — verb (plain dict form) + ですか mixes registers. Casual: いくの？. Polite: 行きますか。
✗ どこへ いきますか？ (using ですか for casual context)
✓ どこへ いくの？
Casual sentence-final の (often with rising intonation) replaces ですか in informal contexts. The plain form いく + の signals casual register. Mixing いきますか with casual context sounds overly formal.

Wrong / corrected pairs

✗ これは あかい くるまです。 (intent: this is the red one)
✓ これは あかいのです。
For pronominal "the red one" use のです (の as pronoun + copula).
✗ たかいの は です。
✓ たかいのは これです。
のは links to a copula; need an explicit subject after は.
✗ あかいですのを かいました。
✓ あかいのを かいました。
Bare adjective + のを (drop です): あかい+の+を.
