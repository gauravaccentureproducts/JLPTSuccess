こう / そう / ああ / どう

ENこう / そう / ああ / どう - 'this way / that way / how'

JA「こう-シリーズ」(ようす・ほうほう)

Explanation

The こう-series describes manner: 'this way/that way/how'. Used adverbially with verbs.

Attaches to

adverbial

Examples

JAこう してください。
ENPlease do it this way.
JAそう おもいます。
ENI think so.
JAどう 行きますか。
ENHow do we get there?
JAこの かんじは どう 読みますか。
ENHow do you read this kanji?
JAこう やりましょう。
ENLet's do it this way.
JAああ しましょう。
ENLet's do it that way.
JAどう おもいますか。
ENWhat do you think?
JAこう しましょう。
ENLet's do it this way.
JAどう しましたか。
ENWhat happened?
JAわたしも そう おもいます。
ENI think so too.

Common mistakes

✗ こうの感じです。
✓ こんな感じです。 / こう思います。
こう/そう/ああ/どう modify verbs ('this way' / 'how') and never take の. To modify a noun use こんな; to modify a verb use こう.
✗ こんな してください。
✓ こう してください。
こう is the ADVERB ('like this' / 'in this way') — modifies the verb する. こんな is the NOUN-MODIFIER ('this kind of') — needs a following noun. For 'do it like this', use こう + する. For 'this kind of person', use こんな + 人.
✗ こんなに してください。
✓ こう してください。
Adverb modifying する is こう/そう/ああ/どう (manner adverb). こんなに is the DEGREE-INTENSIFIER form ('this much'), which modifies adjectives or quantities, not 'how to do'. For 'do it like this', use こう.

Wrong / corrected pairs

✗ どうに します。
✓ どう しますか。
どう as manner-adverb takes no particle. Direct どう + V.
✗ こうな ペン。
✓ この ペン。 / こんな ペン。
こう is an adverb (manner), not a determiner. For "this kind of pen" use こんな or この.
✗ そうします、たべます。
✓ そうします。 / そう、たべます。
そうします itself is a complete response ("I'll do that"). Stacking with another verb is redundant.
