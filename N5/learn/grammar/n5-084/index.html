な-Adjective + な + Noun

ENな-Adjective + な + Noun

JAな-けいようし + な + めいし

Explanation

な-adjectives need な before a noun. しずかな へや = 'a quiet room'. Contrast with い-adjectives, which attach directly (no な): しろい へや = 'a white room'. The な only appears in attributive position — predicate form drops it (へやは しずか).

Attaches to

before_noun

Examples

JAしずかな へやが すきです。
ENI like quiet rooms.
JAゆうめいな 人に あいました。
ENI met a famous person.
JAきれいな はなが さきました。
ENPretty flowers bloomed.
JAしんせつな 先生は すきです。
ENI like kind teachers.
JAわたしの へやは 大きいです。
ENMy room is big.
JA日曜日に へやを そうじします。
ENI clean my room on Sundays.
JAこの へやは ひろいです。
ENThis room is spacious.
JAしずかな まちです。
ENIt's a quiet town.
JAきれいな はなが あります。
ENThere are beautiful flowers.
JAしんせつな ひとに あいました。
ENI met a kind person.

Common mistakes

✗ しずか へや
✓ しずかな へや
な-adj needs な before nouns. (Drop な before です: しずかです.)
✗ しずか へやが すきです。
✓ しずかな へやが すきです。
な-adjectives REQUIRE な before a noun. しずか + な + へや. Without な, the sentence is structurally incomplete at N5. Contrast: い-adjectives skip な (高い 山, not 高いな 山).
✗ しずか へやが すきです。
✓ しずかな へやが すきです。
な-adjective MUST take な before a noun (しずかな へや). The bare form しずか + 名詞 is structurally incomplete. Memorize: な-adj + な + Noun.

Wrong / corrected pairs

✗ しずか へや。
✓ しずかな へや。
な-Adj requires な before noun. Without な it is ungrammatical for noun-modification.
✗ しずかい へや。
✓ しずかな へや。
しずか is na-Adj; do not add い. (い-Adj would be しずか-form like しずかい — does not exist.)
✗ きれいなな ひと。
✓ きれいな ひと。
Single な only. Doubling な is a typo.
