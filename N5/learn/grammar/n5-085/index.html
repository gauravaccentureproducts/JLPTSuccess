な-Adjective + です

ENな-Adjective + です - polite present affirmative

JAな-けいようし + です

Explanation

な-adjectives describe like adjectives but conjugate like nouns: + です for predicative use (このへやは しずかです), + な before a noun (しずかな へや). Common な-adjs: しずか (quiet), きれい (pretty/clean), にぎやか (lively), ゆうめい (famous), すき (like), きらい (dislike).

Attaches to

na_adjective

Common mistakes

✗ きれいい
✓ きれい
きれい LOOKS like an い-adjective (ends in い) but is actually a な-adjective. So きれいな はな (with な) and きれいです (no い-conjugation).
✗ しずかな です。
✓ しずかです。
Drop な before です. な only appears before a noun.
✗ この へやは しずかだ。
✓ この へやは しずかです。
Polite form of な-adjective predicate uses です (しずかです). Plain form drops です and uses だ (しずかだ). Mid-sentence register-switching (starting polite, ending plain) is a beginner tell — pick one register and maintain throughout.

Wrong / corrected pairs

✗ しずかなです。
✓ しずかです。
な only appears between na-adj and a noun (しずかなへや); not at sentence-end with です.
✗ しずかいです。
✓ しずかです。
な-Adj attach directly to です with no い: しずか+です (do not insert い).
✗ きれく ありません。
✓ きれいじゃありません。
きれい LOOKS like an い-adj but is na-Adj: negative is じゃありません, not くありません.
