Verb-stem + たいです

ENVerb-stem + たいです - want to do

JA「〜たいです」で、したい ことを いいます。「水が のみたい」「日本に 行きたいです」。

Explanation

Drop ます, add たい to express the speaker's desire to do something. たい inflects as an い-adjective (negative たくない, past たかった). Object can take が (preferred) or を.

Attaches to

verb_stem_i

Examples

JAにほんへ 行きたいです。
ENI want to go to Japan.
JAコーヒーが 飲みたいです。
ENI want to drink coffee.
JAなにを 食べたいですか。
ENWhat do you want to eat?
JAもう ねたいです。
ENI want to sleep already.
JAきょうは どこへも 行きたくないです。
ENI don't want to go anywhere today.
JAなにを 買いたいですか。
ENWhat do you want to buy?
JAどこに 行きたいですか。
ENWhere do you want to go?
JAすしを たべたいです。
ENI want to eat sushi.
JAにほんに いきたいです。
ENI want to go to Japan.
JAなにが のみたいですか。
ENWhat do you want to drink?

Common mistakes

✗ わたしは あの 人は 行きたいです。
✓ わたしは 行きたいです。
たい expresses the SPEAKER'S desire only - don't use it for third parties. For others, use 〜たがっている.
✗ たべたい です じゃありません
✓ たべたくないです
Negative inflects the い-adj part: drop い, add くない.
✗ にほんに 行くたいです。
✓ にほんへ 行きたいです。
〜たい attaches to the verb STEM (drop ます from ます-form): 行く → 行きます → 行き + たい = 行きたい. Plus: directional motion ('go TO Japan') uses へ (or に) as the destination marker. Stack: 行きたい (want to go) → 行きたいです (polite).
✗ にほんへ 行るたいです。
✓ にほんへ 行きたいです。
〜たい attaches to V-stem (drop ます from V-ます-form, then attach). 行く → 行きます → 行き + たい = 行きたい. The form 行るたい is wrong — must extract stem properly.

Wrong / corrected pairs

✗ 食べたいます。
✓ 食べたいです。
たい conjugates like an い-adjective (NOT a verb): 食べたい + です.
✗ たべるたいです。
✓ たべたいです。
Drop る from Group-2 stem before たい: 食べる→食べ+たい (not 食べる+たい).
✗ せんせいは すしを たべたいです。
✓ せんせいは すしを たべたがっています。
〜たい reports 1st-person desire only; 3rd-person uses 〜たがっている (observable behavior).
