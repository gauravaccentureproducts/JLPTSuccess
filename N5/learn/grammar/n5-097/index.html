〜と〜と、どちらが〜ですか

ENAとBと、どちらが 〜ですか - 'which is more ~, A or B?' (2-way question)

JA「AとBと、どちらが〜ですか」は AとBの 中で どちらが〜か、と ききます。

Explanation

Standard 2-way comparison question form. Pair 2 nouns with と...と, end with どちらが + adj/preference.

Attaches to

noun

Examples

JAいぬと ねこと、どちらが すきですか。
ENWhich do you like more, dogs or cats?
JAコーヒーと おちゃと、どちらが いいですか。
ENCoffee or tea - which is good?
JAなつと ふゆと、どちらが すきですか。
ENWhich do you like better, summer or winter?
JA日本語と えいごと、どちらが むずかしいですか。
ENWhich is more difficult, Japanese or English?
JAりんごと バナナと、どちらが おいしいですか。
ENApple or banana, which is more delicious?
JAねこと いぬと、どちらが すきですか。
ENCat or dog, which do you like?
JAバスと でんしゃと、どちらが はやいですか。
ENBus or train, which is faster?
JAあさと よると、どちらが いいですか。
ENMorning or evening, which is better?
JAやまと うみと、どちらが すきですか。
ENMountain or sea, which do you like?
JAあかいのと あおいのと、どちらが いいですか。

Common mistakes

✗ AかBか、どちらが すきですか。
✓ AとBと、どちらが すきですか。
For "which of A or B?", use と...と to list, then どちら. か...か is grammatical but と...と is the standard frame for comparison questions.
✗ いぬや ねこと、どちらが すきですか。
✓ いぬと ねこと、どちらが すきですか。
Two-way comparison uses と...と (not や). や is for non-exhaustive lists ('apples, bananas, etc.'). When choosing between exactly two items, both joined by と and the question asks どちらが.
✗ いぬか ねこか、どちらが すきですか。
✓ いぬと ねこと、どちらが すきですか。
Two-way comparison joins items with と (both items, then どちら chooses). か would suggest one-or-the-other in a disjunctive sense, but the と...と...どちら frame is fixed for comparing-which-do-you-prefer questions.

Wrong / corrected pairs

✗ コーヒーや こうちゃ、どちらが すきですか。
✓ コーヒーと こうちゃと、どちらが すきですか。
For binary choice, both items are linked by と (exhaustive); や is non-exhaustive list.
✗ A と B と、どっちが すきですか。 (in formal context)
✓ A と B と、どちらが すきですか。
どっち is casual; どちら is polite. Match register.
✗ コーヒーと こうちゃは、どちらが すきですか。
✓ コーヒーと こうちゃと、どちらが すきですか。
Second item also takes と (ABと、 not ABは、) when phrasing the binary choice.
