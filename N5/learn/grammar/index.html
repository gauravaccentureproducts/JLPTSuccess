Copula and Basic Sentence Structure

〜です／〜ます — Polite copula / polite verb ending

Particles

は — Topic marker - introduces what the sentence is about (already-known information) が — Subject marker - marks new information, focus, or the subject of certain stative を — Direct object marker - marks the noun receiving the action of a transitive verb に — Multi-purpose: location of existence / specific time / direction-destination / i へ — Direction marker - toward (used with movement verbs) で — Location of action / means / instrument - where action takes place, by what mean と — With (companion) / and (exhaustive listing) / quotation marker から — From (starting point in time / place) / because (reason) まで — Until / up to (a point in time or place); often paired with から や — And - non-exhaustive listing (these and others) も — Also / too / even (with negation: 'not even / no one / nothing') から〜まで — から 〜 まで - from X to Y (paired range marker) か — Sentence-final question marker か — Or (between alternatives) ね — ね - sentence-final confirmation / agreement particle よ — よ - sentence-final assertion / new-information particle よね — よね - combined: assertion + seeking agreement ("...right?") 〜の — 〜の (possessive / noun-modifier) possessive / noun-modifier — Possessive / noun-modifier nominalizer in basic use — 〜の (nominalizer - turns a verb/adj clause into a noun) 〜の — Sentence-final の as informal question marker (plain-form + の?) だけ — だけ - only / just (limiting particle) しか〜ない — しか〜ない - only (with negative verb) ぐらい / くらい — ぐらい / くらい - about, approximately (used with quantities) ごろ — ごろ - around, approximately (used with time points) など — など - etc. (and so on) ずつ — ずつ - each / per (distributive amount)

Verbs - Tense and Politeness (ます-form)

Existence and Possession

(2-way question) 〜の中で〜が いちばん — Among X (group), Y is the most [adj] — N5 superlative pattern.
