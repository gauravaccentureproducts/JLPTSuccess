に

ENMulti-purpose: location of existence / specific time / direction-destination / indirect object (recipient)

JAばしょ・じかん・人を しめす

Explanation

に has four core uses at N5: (1) location of EXISTENCE with ある／いる, (2) specific point in TIME (7時に, げつようびに), (3) DESTINATION with movement verbs (いく, くる, かえる), (4) RECIPIENT of giving / sending verbs (あげる, おくる, でんわする). Compare carefully with で (location of ACTION) and へ (purely directional).

Attaches to

noun

Examples

JA7時に おきます。
ENI get up at 7 o'clock.
JAにほんに いきません。
ENI'm not going to Japan.
JAへやに なにが ありますか。
ENWhat's in the room?
JAともだちに ほんを あげました。
ENI gave a book to my friend.
JAうちに かえります。
ENI'm going home.
JAねこは いすに います。
ENThe cat is on/by the chair.
JA父に てがみを かきます。
ENI write a letter to my father.
JAがっこうに いきます。
ENI go to school.
JAはちじに おきました。
ENI got up at 8.
JAともだちに あいました。
ENI met (with) my friend.

Common mistakes

✗ としょかんに ほんを よみます。
✓ としょかんで ほんを よみます。
Location of an ACTION (where you do something) takes で. Location of EXISTENCE takes に. Reading is an action, so use で.
✗ あした に いきます。
✓ あした いきます。
Relative time words (あした, きょう, きのう, いま, まいにち) DON'T take に. Only specific clock / calendar times do (7時に, げつようびに, 4月に).
✗ 7時で おきます。
✓ 7時に おきます。
Specific clock times take に (に for absolute/point-in-time markers). で marks the LOCATION of an action (としょかんで べんきょうします), not the TIME. Don't confuse: で = where; に = when (for specific clock times / dates / days-of-week).

Wrong / corrected pairs

✗ 本は つくえで あります。
✓ 本は つくえに あります。
に marks location of existence; で marks location of action.
✗ がっこうで いきます。
✓ がっこうに いきます。
で marks the location of an action; に marks destination of motion.
✗ しちじから おきます。
✓ しちじに おきます。
Specific clock-time takes に (じてん); から marks "starting from".
