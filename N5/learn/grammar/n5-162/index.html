V-plain + まえに

ENVerb-plain まえに - see 〜まえ

JAどうし ふつうけい（じしょけい）＋ まえに。「ねる まえに 本を よみます」。

Explanation

まえ = 'before'. Noun + の + まえ(に); Verb-plain (NON-PAST) + まえに. Note: verb stays NON-PAST even when the meaning is 'before doing X (in the past)'.

Examples

JA出かける まえに、しんぶんを 読みます。
ENBefore going out, I read the newspaper.
JAねる まえに、はを みがきます。
ENBefore sleeping, I brush my teeth.
JA出かける まえに かさを もちます。
ENI take an umbrella before going out.
JAねるまえに はを みがきます。
ENI brush my teeth before going to sleep.
JAごはんの まえに てを あらいます。
ENI wash my hands before meals.
JA三日 まえに 来ました。
ENI came three days ago.
JAとの まえに ねこが います。
ENThere's a cat in front of the door.
JAたべるまえに、 てを あらいます。
ENBefore eating, I wash my hands.
JAねるまえに、 ほんを よみます。
ENBefore sleeping, I read a book.
JAでかけるまえに、 でんわします。
ENBefore going out, I'll call.

Common mistakes

✗ 食べたまえに手を洗います。
✓ 食べるまえに手を洗います。
Verb + まえに requires the DICTIONARY form (plain non-past), regardless of the tense of the main clause. Beginners use 〜たまえに mirroring 'before I ate', which is ungrammatical in Japanese.
✗ 出かけるの まえに、しんぶんを 読みます。
✓ 出かける まえに、しんぶんを 読みます。
V-plain + まえに drops の (no NOUN+の structure). The form 出かけるの まえに mixes the V-form with the noun-form. Pattern: V-dictionary + まえに (no の). 食べる まえに、本を読む。
✗ 出かけるの まえに、しんぶんを 読みます。
✓ 出かける まえに、しんぶんを 読みます。
V-plain + まえに: drop の. The pattern V-dict + まえに is for 'before V-ing'. The noun-pattern (Noun + の + まえに) has の; the V-pattern doesn't. Different rules for different word classes.

Wrong / corrected pairs

✗ たべますまえに。
✓ たべるまえに。
まえに takes plain dictionary form, not polite ます.
✗ たべたまえに。
✓ たべるまえに。
まえに takes NON-PAST plain (Vる); past form + まえに is wrong.
✗ たべるまえで、 ねます。
✓ たべるまえに、 ねます。
Time-before takes に, not で.
