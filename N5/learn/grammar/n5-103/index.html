〜ができます

EN〜が できます - can do / be able to / have ability in

JA「〜が できます」で できる ことを いいます。「日本ごが できます」。

Explanation

できる = 'can do, be able to'. Subject (skill / language / activity) takes が. Also 'be ready/finished'.

Attaches to

noun

Common mistakes

✗ 日本語を できます。
✓ 日本語が できます。
できる ('can do, be capable') is an adjectival predicate that treats the ability-target as the SUBJECT of the can-ness, not a direct object. So 日本語が できます (Japanese is do-able for me), not 日本語を できます. Same が-rule as すき / じょうず / わかる / ほしい.
✗ Treating できる as a single "can" verb
✓ できる has two senses: capability (日本語ができる = "I am able with Japanese") AND completion/readiness (ばんごはんができた = "Dinner is ready"). Context distinguishes which sense applies.
A learner who only knows the "can do" sense will misread 「ばんごはんができました」 as "I can make dinner" (wrong). The completion sense is essential for restaurants, schedules, and announcements.
✗ 日本語を できます。
✓ 日本語が できます。
できる ('can do, be capable') takes が. The ability-object goes with が, not を. For 'I can speak Japanese', 日本語が できます. The same が-rule applies to all adjectival-predicate verbs in this family.

Wrong / corrected pairs

✗ にほんごを できます。
✓ にほんごが できます。
できる takes が, not を. (Like すき/上手 — feeling/capability adjectives use が.)
✗ たべるが できます。
✓ たべることが できます。
For "can do (verb)" use Vる + ことが できる. Bare Vる + が cannot directly precede できる.
✗ すこし にほんご できます。
✓ すこし にほんごが できます。
Object of できる still requires が.
