なにか / なにも

ENSomething / nothing - see compound rule

JA「なにか」は 「ある もの」、「なにも」は 「ぜんぶ ない」と いう いみです。「なにか 食べたい」「なにも 食べない」。

Explanation

Pattern: question word + か = 'some-' (indefinite). + も + affirmative = 'every-'. + も + negative = 'no-'.

Examples

JAなにか たべたいです。
ENI want to eat something.
JAなにも たべません。
ENI won't eat anything.
JAなにか のみますか。
ENWill you have something to drink?
JAきょうは なにも しません。
ENI won't do anything today.
JAなにか 食べましょう。
ENLet's eat something.
JAなにも 言いませんでした。
ENI said nothing.
JAなにか 飲みたいです。
ENI want to drink something.
JAなにか のみたいです。
ENI want to drink something.
JAなにも たべませんでした。
ENI didn't eat anything.
JAなにか いいことが ありましたか。
ENDid anything good happen?

Common mistakes

✗ なにか 食べません。
✓ なにも 食べません。
なにか pairs with affirmative ("eat something"); なにも with negative ("eat nothing"). Don't mix.
✗ なにも 食べたいです。
✓ なにか 食べたいです。
"I want to eat something" is affirmative - use なにか, not なにも.
✗ なにを たべたいです。
✓ なにか たべたいです。
Same as n5-183: なに-か with か (indefinite). For 'I want to eat something', use なにか (no を). Adding を would force なに into question-word mode and create an incomplete sentence.

Wrong / corrected pairs

✗ 何も 食べました。
✓ 何か 食べました。 / 何も 食べませんでした。
何も requires negative verb; 何か requires positive.
✗ なにも たべました。 (intent: I ate something)
✓ なにか たべました。
なにか = "something" (positive); なにも = "nothing" (negative + Vません).
✗ なにも たべません か。
✓ なにも たべませんでした。 / なにか たべましたか。
なにも requires negative predicate; cannot pair with question か meaning "did you eat anything".
