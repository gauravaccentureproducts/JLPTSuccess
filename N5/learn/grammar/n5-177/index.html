Verb-stem / Adj-stem + すぎる

ENVerb-stem / Adj-stem + すぎる - too much / excessive

JA「〜すぎる」

Explanation

Drop ます/い, add すぎる. 'Too much / too X'. Examples: 食べすぎる ('eat too much'), 高すぎる ('too expensive'), しずかすぎる ('too quiet'). Conjugates as a regular る-verb (ichidan): すぎます, すぎました, すぎない.

Attaches to

verb_or_adj_stem

Examples

JAきのう たべすぎました。
ENI ate too much yesterday.
JAこの りょうりは あますぎます。
ENThis food is too sweet.
JAこの くつは 大きすぎます。
ENThese shoes are too big.

ENI ate too much yesterday.

ENI ate too much today.

ENThis dish is too spicy.
JAたかすぎて 買えません。
ENIt's too expensive to buy.

Common mistakes

✗ 食べてすぎました。
✓ 食べすぎました。
〜すぎる attaches to verb-STEM (the ます-form minus ます), not to the て-form. 食べる -> 食べ + すぎる -> 食べすぎる. Adjectives use the root: 高い -> 高すぎる; 静か -> 静かすぎる.
✗ きのう たべるすぎました。
✓ きのう たべすぎました。
すぎる attaches to verb STEM (drop ます from ます-form). たべる → たべます → たべ + すぎる = たべすぎる. The form たべるすぎる is wrong — must extract stem. Pattern: V-stem + すぎる.
✗ きのう たべるすぎました。
✓ きのう たべすぎました。
〜すぎる attaches to V-stem (drop ます). たべる → たべます → たべ + すぎる = たべすぎる. The form たべるすぎ wrongly keeps dict-form る. Pattern: V-stem + すぎる.

Wrong / corrected pairs

✗ たべるすぎる。
✓ たべすぎる。
すぎる attaches to V-stem (drop る from G2): 食べる→食べ+すぎる.
✗ たかいすぎる。
✓ たかすぎる。
い-Adj drops い before すぎる: 高い→高+すぎる.
✗ しずかいすぎる。
✓ しずかすぎる。
na-Adj uses bare stem + すぎる (no な, no い). しずか→しずか+すぎる.
