けれど / けど

ENけれど / けど - 'but' (informal contrast)

JA「けれど／けど」は 「が」「しかし」と おなじいみで、はなしことばで よく つかいます。「行きたい けど、いそがしい」。

Explanation

Casual equivalent of が (clause-final 'but'). けれど is slightly more formal than けど.

Attaches to

clause_end

Examples

JAむずかしいですけど、おもしろいです。
ENIt's hard, but interesting.
JA行きたいけど、時間が ありません。
ENI want to go, but I don't have time.
JA日本語は たのしいけど、かんじが むずかしい。
ENJapanese is fun, but kanji is hard.
JAやすいけど、おいしくないです。
ENIt's cheap, but it's not tasty.
JAいきたいけど、時間が ありません。
ENI want to go, but I have no time.
JAすきだけど、たかすぎます。
ENI like it, but its too expensive.
JAべんきょうしたけれど、わすれました。
ENI studied, but I forgot.
JAたかいけど、 おいしい。
ENIt's expensive, but tasty.
JAがっこうは とおいけど、 たのしい。
ENSchool is far, but fun.
JAにほんごは むずかしいけれど、 おもしろい。
ENJapanese is difficult, but interesting.

Common mistakes

✗ 行きましたけど、楽しいでした。
✓ 行きましたけど、楽しかったです。
けど/けれど doesn't fix the conjugation of the second clause - past-tense i-adjectives still take the い→かった ending (楽しかった), not 楽しいでした. Beginners use けど as a 'reset' and forget conjugation rules.
✗ むずかしいです けれども、おもしろいです。
✓ むずかしいですけど、おもしろいです。
Both けれども (this 'wrong' form) and けど (the 'correct' contraction) are valid. けれども is the FULL form (more formal / written); けど is the casual contraction. Register choice, not a grammatical error.
✗ むずかしいですけど、おもしろい。 (mixed registers)
✓ むずかしいですけど、おもしろいです。
Polite register throughout: both clauses end in です. Mixing polite (むずかしいです) with plain (おもしろい) creates inconsistent register. Pick one and maintain.

Wrong / corrected pairs

✗ たかい けど、おいしい。 (in formal context)
✓ たかい けれど、おいしい。 / たかい です が、おいしい です。
けど is the casual form of けれど; use けれど (or が) in formal context.
✗ たかいです けど です。
✓ たかい けど、おいしい。 / たかい ですが、おいしい です。
けど takes plain form on the left side. Mixing です + けど + です is non-standard.
✗ たかい けれど、しかし おいしい。
✓ たかい けれど、おいしい。
けれど + しかし stacks two contrastive markers; pick one.
