〜まえ

EN〜まえ - before

JA「〜まえ」は ある ことの まえを いいます。「ごはんの まえに 手を あらいます」。

Explanation

まえ = 'before'. Noun + の + まえ(に); Verb-plain (NON-PAST) + まえに. Note: verb stays NON-PAST even when the meaning is 'before doing X (in the past)'.

Attaches to

noun

Examples

JAごはんの まえに てを あらいます。
ENI wash my hands before meals.
JAねる まえに 本を 読みます。
ENI read a book before going to bed.
JAテストの まえに よく べんきょうしました。
ENI studied well before the test.
JAばんごはんの まえに しゅくだいを します。
ENI do my homework before dinner.
JA三日 まえに 来ました。
ENI came three days ago.
JAがっこうの まえに ともだちが います。
ENMy friend is in front of the school.
JA五分まえに 来ました。
ENI came 5 minutes before.
JAねるまえに、 はを みがきます。
ENI brush my teeth before sleeping.
JAがっこうの まえに、 あさごはんを たべます。
ENI eat breakfast before school.
JAたべるまえに、 てを あらいます。
ENI wash my hands before eating.

Common mistakes

✗ ねたまえに
✓ ねるまえに
Use plain non-past before まえに - even for past actions.
✗ ごはん まえに てを あらいます。
✓ ごはんの まえに てを あらいます。
Noun + の + まえに = 'before [noun]'. Don't drop の. Pattern: NOUN + の + まえに + Verb. For V-plain + まえに ('before doing'), drop の: 食べる まえに.
✗ ごはん まえに てを あらいます。
✓ ごはんの まえに てを あらいます。
Noun + の + まえに = 'before [noun]'. Don't drop の. Pattern: NOUN + の + まえに + Verb. For V-plain + まえに (before doing), drop の: 食べる まえに.

Wrong / corrected pairs

✗ たべますまえに、 てを あらいます。
✓ たべるまえに、 てを あらいます。
まえに takes plain dictionary form, not polite ます.
✗ たべるまえで、 てを あらいます。
✓ たべるまえに、 てを あらいます。
まえに (time before) — particle is に, not で.
✗ にじまえに きます。
✓ にじのまえに きます。
For "before 2 o'clock" use にじのまえに (with の), or にじまえに with hyphenation context.
