だれ / どなた

ENWho

JA「だれ」(ふつう) ／ 「どなた」(ていねい)

Explanation

だれ is the standard 'who'. どなた is more polite - used with strangers, customers, or in formal settings. Question words asking for new information take が, not は: だれが きましたか, never だれは.

Examples

JAだれが きましたか。
ENWho came?
JAこれは だれの かばんですか。
ENWhose bag is this?
JAどなたですか。
ENWho is it (polite)?
JAだれと いきますか。
ENWith whom are you going?
JAあの 人は だれですか。
ENWho is that person?
JAだれの ほんですか。
ENWhose book is this?
JAどなたが せんせいですか。
ENWho is the teacher?
JAあの ひとは だれですか。
ENWho is that person?
JAどなたが やまだ先生ですか。
ENWho is the teacher Yamada?
JAこれは だれの ですか。
ENWhose is this?

Common mistakes

✗ 誰さんですか。
✓ 誰ですか。 / どなたですか。
誰 (だれ) is a question word and does not take the honorific suffix さん. The polite version is どなた (a separate word). Mixing 誰 + さん sounds infantile or sarcastic.
✗ だれは きましたか。
✓ だれが きましたか。
Question word in subject position takes が, never は. だれは would imply 'as for [who]' — but the very point is we don't know who. The answer also uses が: '田中さんが きました'.
✗ だれですか きましたか。
✓ だれが きましたか。
Question-word subject takes が — directly before the verb, not as a separate copula clause. 'だれです' would be a noun-predicate sentence ('who is it?'); for 'who came', use だれ + が + 来ました + か.

Wrong / corrected pairs

✗ だれの くるまは どれですか。
✓ どれが だれの くるまですか。
Question word in subject position takes が. Reorder so the unknown takes が.
✗ やまださんは だれ ですか。 (asking who Yamada is, formal)
✓ やまださんは どなた ですか。 / やまださんは どんな ひと ですか。
For "who is X" formally use どなた; informally だれ.
✗ だれは いますか。
✓ だれが いますか。
Q-word だれ takes が, never は.
