Verb-stem + たくないです

ENVerb-stem + たくないです - don't want to

JA「〜の 中で〜が いちばん〜」で、何かの 中で いちばん〜だ、と いいます。「くだものの 中で りんごが いちばん すきです」。

Explanation

Negative of たい - drop い from たい, add くない (treats たい as い-adj).

Attaches to

verb_stem_i

Examples

JA今日は 行きたくないです。
ENI don't want to go today.
JAなにも たべたくないです。
ENI don't want to eat anything.
JAびょういんに 行きたくないです。
ENI don't want to go to the hospital.
JAしゅくだいを したくないです。
ENI don't want to do homework.
JAあした しごとに 行きたくないです。
ENI don't want to go to work tomorrow.
JAむずかしい もんだいは したくないです。
ENI don't want to do difficult problems.
JAきょうは いきたくないです。
ENI don't want to go today.
JAいまは なにも たべたくないです。
ENI don't want to eat anything now.
JAがっこうに いきたくないです。
ENI don't want to go to school.
JAはたらきたくないです。
ENI don't want to work.

Common mistakes

✗ 行きたくありません。 (correct but rigid)
✓ 行きたくないです。 (more natural)
Both 〜たくありません and 〜たくないです are grammatically correct polite negative volitional forms. 〜たくないです is slightly more natural in modern conversational Japanese; 〜たくありません sounds slightly more formal / older-register but is NOT wrong.
✗ 食べたくない です。
✓ 食べたくないです。
たくない is one unit; no space between くない and です.
✗ 今日は 行かないたいです。
✓ 今日は 行きたくないです。
〜たい conjugates like an い-adjective for negation: drop い from たい, add くない → たくない. The form 行かないたい (negative form of verb + たい) is structurally wrong — first form たい (positive), then negate as if it were an adjective: 行きたい → 行きたくない.
✗ 今日は 行るたくないです。
✓ 今日は 行きたくないです。
〜たい + negative: drop い from たい, add くない (i-adj negation pattern). Attach to V-stem. Pattern: V-stem + たくない. The form 行るたくない wrongly keeps the dict-form る.
✗ 今日は 行きたくないでした。
✓ 今日は 行きたくなかったです。
〜たい conjugates like i-adjective for past: drop い from たくない, add かった → たくなかった. Then add です for polite. Don't add でした after たくない.

Wrong / corrected pairs

✗ 食べたいません。
✓ 食べたくないです。 / 食べたくありません。
たい negative is たくない (i-adj rule), NOT verb-style.
✗ たべたいなくないです。
✓ たべたくないです。
たい conjugates like an い-Adj: negative is たくない (drop い, add くない).
✗ たべないたいです。
✓ たべたくないです。
Order matters: stem + たい first, then negate the い-Adj. Not Vない + たい.
