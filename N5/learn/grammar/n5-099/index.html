〜がすきです / きらいです

EN〜が すきです / きらいです - like / dislike

JA「〜が すきです／きらいです」で すきな ものや きらいな ものを いいます。

Explanation

すき / きらい are な-adjectives. The thing liked/disliked takes が, NEVER を. Intensified forms: だいすき = 'love' / だいきらい = 'hate' (literally 'big like / big dislike'). Negation: すきじゃない / きらいじゃない.

Attaches to

noun

Examples

JAわたしは ねこが すきです。
ENI like cats.
JAなっとうが きらいです。
ENI dislike natto.
JAなにが すきですか。
ENWhat do you like?
JAにくは すきですが、さかなは きらいです。
ENI like meat, but I dislike fish.
JAわたしは いぬが きらいです。
ENI dislike dogs.
JAあまい ものが すきです。
ENI like sweet things.
JAどんな りょうりが すきですか。
ENWhat kind of dishes do you like?
JAわたしは すしが すきです。
ENI like sushi.
JAスポーツが あまり すきじゃ ありません。
ENI don't really like sports.
JAどんな おんがくが すきですか。
ENWhat kind of music do you like?

Common mistakes

✗ ねこを すきです。
✓ ねこが すきです。
Stative predicates (すき, きらい, わかる, できる) take が, never を.
✗ わたしは ねこを すきです。
✓ わたしは ねこが すきです。
すき / きらい / じょうず / へた / ほしい / わかる are ADJECTIVAL predicates that treat their object as the SUBJECT of feeling, not a direct object. So they take が, not を. The を-form is a common beginner error — to like / dislike / understand a thing, that thing is が.
✗ わたしは ねこを すきです。
✓ わたしは ねこが すきです。
Same as n5-098 — すき is the family marker for this construction. Liking, ability, possession-feeling all take が. Even though English uses 'I LIKE cats' transitively, Japanese treats 'cats' as the subject of the liking-state.

Wrong / corrected pairs

✗ 私は コーヒーが すきます。
✓ 私は コーヒーが すきです。
すき is a na-adjective; takes です, not ます (which is for verbs).
✗ わたしは ねこを すきです。
✓ わたしは ねこが すきです。
すき/きらい take が, not を. The "object" of liking is grammatically the subject.
✗ すきくないです。
✓ すきじゃありません。
すき is na-Adj: negative uses じゃありません, not くないです.
