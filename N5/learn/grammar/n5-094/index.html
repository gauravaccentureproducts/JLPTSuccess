〜があります

EN〜が あります - 'have / there is' (events, skills, possessions)

JA「〜が あります」(イベント・スキル)

Explanation

Beyond physical existence, あります covers events ('there's an exam'), abstract things ('time / money'), and possession ('I have...'). Use が for the subject.

Attaches to

noun_subject

Examples

JAあした しけんが あります。
ENThere's an exam tomorrow.
JAおかねが ありません。
ENI don't have money.
JAなにか しつもんが ありますか。
ENDo you have any questions?
JAあした じゅぎょうが ありません。
ENTomorrow there are no classes.
JAここに ペンが あります。
ENThere is a pen here.
JAここに たまごが あります。
ENThere are eggs here.
JAつくえの 上に ノートが あります。
ENThere is a notebook on the desk.
JAにもつが たくさん あります。
ENThere is a lot of luggage.
JAしつもんが ありますか。

JAたいせつな ものが あります。

Common mistakes

✗ テストは あした います。
✓ あした テストが あります。
For events/scheduled things (test, meeting, party), use 〜が あります. The subject is the event, not its time.
✗ あした しけんが います。
✓ あした しけんが あります。
しけん (an exam — abstract event/schedule) uses あります. Even though events involve people, the event ITSELF is non-animate. Use あります for events, schedules, plans, possessions, abstract concepts; います only for animate living beings.
✗ あした しけんが ます。
✓ あした しけんが あります。
Full verb form is あります (polite present of ある). Don't truncate to just ます — that's the polite-form suffix, not a verb. Pattern: NOUN + が + あります.

Wrong / corrected pairs

✗ うちに ねこが あります。
✓ うちに ねこが います。
Animate → います.
✗ ほんが あった。 (about future possession)
✓ ほんが あります。
あった is plain past; non-past plain is ある, polite is あります.
✗ にほんへ あります。 (intent: have something in Japan)
✓ にほんに あります。
Static location takes に; へ is direction.
