Verb-た

ENPlain past affirmative (た-form)

JAふつうの かこけい

Explanation

た-form (plain past) follows the same sound-change rules as the て-form, just with た instead of て. Memorize the group rules - they're foundational for advanced grammar (たことがある, たほうがいい, etc.).

verb_root

Common mistakes

✗ いく → いいた
✓ いく → いった
いく is the standard exception: it takes -った, not -いた like other -く verbs. Memorize this one.
✗ きのう ほんを よみた。
✓ きのう ほんを よんだ。
Group-1 plain past for verbs ending in む: change む to ん, add だ. よむ → よん + だ = よんだ. The form よみた applies the wrong sub-rule. Memorize: む/ぶ/ぬ → んだ; く → いた; ぐ → いだ; す → した; つ/る/う → った.
✗ きのう ほんを よみた。
✓ きのう ほんを よんだ。
Group 1 む-ending verbs: past plain uses んだ (む → んだ). よむ → よんだ. The form よみた applies the wrong sub-rule. Memorize the 5 Group-1 sub-rules: く→いた, ぐ→いだ, す→した, つ・る・う→った, ぶ・む・ぬ→んだ.

Wrong / corrected pairs

✗ 行いた。
✓ 行った。
行く has irregular te/ta-form: 行く → 行って / 行った.
✗ いきた。
✓ いった。
行く is a Group-1 exception: 行って/行った (NOT 行きて/行きた). Stem ends -く but uses っ-promotion.
✗ べんきょうた。
✓ べんきょうした。
する→した, not 〜た. Compound する-verbs all behave this way.
