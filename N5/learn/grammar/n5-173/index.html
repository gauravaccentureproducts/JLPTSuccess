〜なくてはいけない

EN〜なくては いけない - must do

JA「〜なくては いけない」

Explanation

Negative て-form + は + いけない = 'must do' (lit. 'not doing won't work'). Casual contraction: 〜なくちゃ.

Attaches to

verb_nai

Examples

JAまいにち べんきょうしなくては いけない。
ENI must study every day.
JAはやく かえらなくては いけません。
ENI have to go home early.
JAくすりを 飲まなくては いけない。
ENI must take the medicine.
JA今日 しゅくだいを 出さなくては いけません。
ENI have to submit homework today.
JAまいにち かんじを おぼえなくては いけません。
ENYou have to memorize kanji every day.
JAはやく 学校へ 行かなくては いけません。
ENI have to go to school early.
JAもっと べんきょうしなくては いけません。
ENI have to study more.
JAもう いかなくては いけません。
ENI have to go now.
JAやくそくを まもらなくては いけません。
ENI have to keep my promise.
JAはやく ねなくては いけません。
ENI have to sleep early.

Common mistakes

✗ 行きなくては いけません。
✓ 行かなくては いけません。
〜なくては attaches to the negative-stem (ない-form base): 行か + なくては. NOT to the polite-stem 行き.
✗ まいにち べんきょうしなくは いけない。
✓ まいにち べんきょうしなくては いけない。
Obligation 〜なくては いけない: drop い from ない, add くて, then は. The form なくは (skipping the て) is wrong — the て-form is part of the derivation chain ない → なくて → なくては. Don't shortcut the て.
✗ まいにち べんきょうしないと いけない。
✓ まいにち べんきょうしなくては いけない。
Obligation: V-ない (drop い + くて) + は + いけない → なくては いけない. The form しないと いけない is a SEPARATE construction (ない + と + いけない = 'must do' via different derivation). Both are valid N5 but n5-173 specifically uses なくてはいけない.

Wrong / corrected pairs

✗ たべないて はいけない。
✓ たべなくては いけない。
Negative form is なくて, not ないて.
✗ たべなくちゃ いけない。 (in formal context)
✓ たべなくては いけません。
なくちゃ is casual contraction of なくては; in formal use full form + ません.
✗ たべないと いけない。 (intent: in formal context)
✓ たべなくては いけません。
ないと-いけない and なくては-いけない overlap; なくては is more formal.
