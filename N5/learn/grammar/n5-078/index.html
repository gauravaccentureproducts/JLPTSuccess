い-Adjective + Noun

ENい-Adjective + Noun - adjective directly before noun

JA「い-けいようし + めいし」で めいしを しゅうしょくします。「い-けいようし」は そのまま めいしに つけます(「な」は いりません)。「たかい 山」「あつい 日」「あたらしい 本」。

Explanation

い-adjectives modify nouns directly, no particles needed. Keep the い: 高い 山 = 'tall mountain'.

Attaches to

before_noun

Examples

JA高い 山が 見えます。
ENI can see a tall mountain.
JAおもしろい 本を 読みました。
ENI read an interesting book.
JAあつい コーヒーが すきです。
ENI like hot coffee.
JAたかい 山に のぼりました。
ENI climbed a high mountain.
JAおおきい いえに すんでいます。
ENI live in a big house.
JAたかい くるまを 見ました。
ENI saw an expensive car.
JAあたらしい ふくを 買いました。
ENI bought new clothes.
JAたかい くるまを かいました。
ENI bought an expensive car.
JAちいさい ねこが います。
ENThere is a small cat.
JAあたらしい ほんを よみました。
ENI read a new book.

Common mistakes

✗ 高いの 山
✓ 高い 山
Don't insert の between い-adj and noun. の is for noun-noun linking only.
✗ 高いの 山が 見えます。
✓ 高い 山が 見えます。
い-adjectives attach DIRECTLY to nouns — no の between. The form 高いの would nominalize 高い ('the expensive one'), not modify 山. Pattern: い-adj + Noun (no particle). な-adj uses な.
✗ 高いの 山が 見えます。
✓ 高い 山が 見えます。
い-adjectives modify nouns directly (高い + 山). No の, no な between adjective and noun. Adding の nominalizes the adjective ('the high one'), changing the meaning entirely.

Wrong / corrected pairs

✗ たかいなほん。
✓ たかい ほん。
い-Adj attaches directly to noun (no な). な-Adj uses な; い-Adj doesn't.
✗ おおきの くるま。
✓ おおきい くるま。 / おおきな くるま。
おおき is the stem; full い-Adj is おおきい. The な-form おおきな is a special variant (not all い-Adj have な-version).
✗ いいの ひと。
✓ いい ひと。
い-Adj attaches directly; do not insert の.
